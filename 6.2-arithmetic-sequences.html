6.2 Arithmetic Sequences
Objective: Write nth-term formulas, find arithmetic means, and evaluate finite arithmetic series.

Reference
Arithmetic Sequence Formulas
difference
\[d = a_n - a_{n-1}\]
nth term
\[a_n = a_1 + (n-1)d\]
finite sum
\[S_n = \dfrac{n}{2}(a_1 + a_n)\]
\[S_n = \dfrac{n}{2}\bigl(2a_1 + (n-1)d\bigr)\]
Arithmetic sequences behave like linear functions: the common difference \(d\) acts like the slope.
Example 1
Finding an Explicit Formula
Find an explicit formula for the arithmetic sequence with first term \(66\) and common difference \(-11\).
Use the nth-Term Formula
\[a_n = a_1 + (n-1)d\]
Substitute the Given Values
\[a_n = 66 + (n-1)(-11)\]
Simplify
\[a_n = 66 - 11n + 11 = 77 - 11n\]
Answer
\[a_n = 77 - 11n\]
Example 2
Using Two Known Terms
The 4th term of an arithmetic sequence is \(24\) and the 13th term is \(87\). Write the first five terms.
Find the Common Difference
\[d = \dfrac{87 - 24}{13 - 4} = \dfrac{63}{9} = 7\]
There are \(9\) equal jumps from term 4 to term 13.
Back Up to the First Term
\[a_1 = 24 - 3(7) = 3\]
List the First Five Terms
\[3,\ 10,\ 17,\ 24,\ 31\]
Example 3
Arithmetic Means
Insert three arithmetic means between \(21\) and \(45\).
Count the Equal Jumps
\[21,\ \square,\ \square,\ \square,\ 45\]
Five terms create four equal jumps.
Compute the Common Difference
\[d = \dfrac{45 - 21}{4} = 6\]
Fill in the Means
\[21,\ 27,\ 33,\ 39,\ 45\]
Concept
Finite Arithmetic Series
A finite arithmetic series adds the terms of an arithmetic sequence. Pairing the first and last terms reveals a fast pattern.
Gauss Pairing
\[(1+100),\ (2+99),\ (3+98), \dots\]
Same Sum Every Time
Each pair adds to \(101\).
How Many Pairs?
There are \(50\) pairs, so \(S_{100} = 50(101) = 5050\).
Takeaway
If the first and last terms are easy to find, \(S_n = \dfrac{n}{2}(a_1+a_n)\) is the fastest formula.
Example 4
Finding a Large Partial Sum
Find the 101st partial sum of the arithmetic sequence \(7, 9.5, 12, \dots\).
Identify the Sequence Data
  • \(a_1 = 7\)
  • \(d = 2.5\)
  • \(n = 101\)
Find the 101st Term
\[a_{101} = 7 + 100(2.5) = 257\]
Use the Sum Formula
\[S_{101} = \dfrac{101}{2}(7 + 257) = \dfrac{101}{2}(264) = 13332\]
Example 5
Arithmetic-Series Application
A business sells \(\$20{,}000\) during its first year and increases sales by \(\$15{,}000\) each year for 19 more years. Find the total sales over the first 20 years.
Model the Annual Sales
  • \(a_1 = 20000\)
  • \(d = 15000\)
  • \(n = 20\)
Find the 20th Year
\[a_{20} = 20000 + 19(15000) = 305000\]
Add the First 20 Terms
\[S_{20} = \dfrac{20}{2}(20000 + 305000) = 10(325000) = 3250000\]
Answer
\[\$3{,}250{,}000\]
