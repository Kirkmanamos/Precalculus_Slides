Definition
Arithmetic Sequences
  • An ______________________ arithmetic sequence is one that has a common difference, \(d\).
  • In other words, you ______ add \(d\) to get to the next term.
SequenceCommon Difference
\(1,\ 2,\ 3,\ 4,\ 5,\ 6,\ \dots\)\(d = 1\)
\(0,\ 3,\ 6,\ 9,\ 12,\ \dots\)\(d = 3\)
\(10,\ 5,\ 0,\ {-5},\ {-10},\ \dots\)\(d = -5\)

Slope Idea
For a linear function, slope measures “change in output per 1 step in input.” In an arithmetic sequence, that constant change is the common difference \(d\).
Why the nth-Term Formula Works
Start at \(a_1\). To reach term \(n\), you make exactly \(n-1\) equal jumps of size \(d\), so \[a_n = a_1 + (n-1)d\]
Self-check: if the consecutive differences stay constant, the sequence is arithmetic.

Reference
Arithmetic Sequence Formulas
difference
\[d = a_n - a_{n-1}\]
nth term
\[a_n = a_1 + (n-1)d\]
finite sum
\[S_n = \dfrac{n}{2}(a_1 + a_n)\]
\[S_n = \dfrac{n}{2}\bigl(2a_1 + (n-1)d\bigr)\]
Arithmetic sequences behave like linear functions: the common difference \(d\) acts like the slope.
Which sum formula?
Know the last term? Use \(S_n = \dfrac{n}{2}(a_1 + a_n)\).
Don't know it? Use \(S_n = \dfrac{n}{2}(2a_1 + (n{-}1)d)\).
Why \((n-1)d\)?
The first term is already \(a_1\). From there, it takes \(n-1\) equal jumps of size \(d\) to reach \(a_n\).
Example 1
Finding an Explicit Formula
Find an explicit formula for the arithmetic sequence with first term \(66\) and common difference \(-11\).
Use the nth-Term Formula
\[a_n = a_1 + (n-1)d\]
Substitute the Given Values
\[a_n = 66 + (n-1)(-11)\]
Simplify
\[a_n = 66 - 11n + 11 = 77 - 11n\]
Answer
\[a_n = 77 - 11n\]
Your Turn
Practice: Write an Explicit Formula
Write an explicit formula for the arithmetic sequence with first term \(14\) and common difference \(-3\).
Hint
Start with \(a_n = a_1 + (n-1)d\). Substitute \(a_1 = 14\) and \(d = -3\), then simplify.
Answer
\[a_n = 14 + (n-1)(-3) = 17 - 3n\]
Example 2
Using Two Known Terms
The 4th term of an arithmetic sequence is \(24\) and the 13th term is \(87\). Write the first five terms.
Find the Common Difference
\[d = \dfrac{87 - 24}{13 - 4} = \dfrac{63}{9} = 7\]
There are \(9\) equal jumps from term 4 to term 13.
Back Up to the First Term
\[a_1 = 24 - 3(7) = 3\]
List the First Five Terms
\[3,\ 10,\ 17,\ 24,\ 31\]
Example 2
Alternative Method: Use Two Equations
Use the explicit formula with the given terms \(a_4 = 24\) and \(a_{13} = 87\).
Write Two Equations
\[24 = a_1 + 3d,\qquad 87 = a_1 + 12d\]
Subtract to Solve for \(d\)
\[87 - 24 = (a_1 + 12d) - (a_1 + 3d)\Rightarrow 63 = 9d \Rightarrow d = 7\]
Find the First Term
\[24 = a_1 + 3(7)\Rightarrow 24 = a_1 + 21\Rightarrow a_1 = 3\]
Same Sequence
\[3,\ 10,\ 17,\ 24,\ 31\]
This method is useful when you are given two specific terms and want an equation-based approach.
Your Turn
Practice: Two Known Terms
The 3rd term of an arithmetic sequence is \(10\) and the 7th term is \(26\). Find \(a_1\) and \(d\).
Hint
First find the common difference by comparing term numbers and term values. Then back up from the 3rd term to the 1st term.
Answer
\[d = \frac{26-10}{7-3} = 4,\qquad a_1 = 10 - 2(4) = 2\]
Definition
Arithmetic Means
  • Arithmetic means are the numbers that fit __________ between two given terms of an arithmetic sequence to create a constant ____________ difference .
Example

Two arithmetic means between \(5\) and \(17\):

\[5,\ \underline{\quad 9\quad},\ \underline{\quad 13\quad},\ 17\]
Strategy
Count the total number of terms (including both endpoints), then find \(d = \dfrac{a_{\text{last}} - a_{\text{first}}}{\text{number of jumps}}\).
Why "Means"?
Each inserted value is the average of its two neighbors. For instance, \(9 = \dfrac{5+13}{2}\).
Example 3
Inserting Arithmetic Means
Insert three arithmetic means between \(21\) and \(45\).
Count the Equal Jumps
\[21,\ \square,\ \square,\ \square,\ 45\]
Five terms create four equal jumps.
Compute the Common Difference
\[d = \dfrac{45 - 21}{4} = 6\]
Fill in the Means
\[21,\ 27,\ 33,\ 39,\ 45\]
Example 3
Alternative Method: Use Averages
Insert three arithmetic means between \(21\) and \(45\) by using the idea of a mean.
Find the Middle Mean First
\[\frac{21+45}{2} = 33\]
With five total terms, there is one middle term halfway between the endpoints.
Average the Left Pair
\[\frac{21+33}{2} = 27\]
Average the Right Pair
\[\frac{33+45}{2} = 39\]
Same Means
\[21,\ 27,\ 33,\ 39,\ 45\]
This method works nicely here because there is a single middle term.
Concept
Finite Arithmetic Series
A finite arithmetic series adds the terms of an arithmetic sequence. Pairing the first and last terms reveals a fast pattern.
Takeaway
If the first and last terms are easy to find, \(S_n = \dfrac{n}{2}(a_1+a_n)\) is the fastest formula.
Concept
Arithmetic Sequence vs. Arithmetic Series
An arithmetic sequence is a list of terms with a constant difference. An arithmetic series is the sum of those terms. A partial sum adds only the first \(n\) terms.
Sequence

This is a list:

\[7,\ 9.5,\ 12,\ 14.5,\ \dots\]
Series

This is a sum:

\[7 + 9.5 + 12 + 14.5 + \dots\]
Partial Sum

\(S_{101}\) means:

\[\text{the sum of the first 101 terms}\]
Self-check: the subscript in \(S_n\) tells how many terms are being added, not the value of the last term.
Example 4
Finding a Large Partial Sum
Find the 101st partial sum of the arithmetic sequence \(7, 9.5, 12, \dots\).
Interpret the Question
  • \(S_{101}\) means add the first 101 terms.
  • \(a_1 = 7\)
  • \(d = 2.5\)
  • \(n = 101\)
Find the 101st Term
\[a_{101} = 7 + 100(2.5) = 257\]
Use the Sum Formula
\[S_{101} = \dfrac{101}{2}(7 + 257) = \dfrac{101}{2}(264) = 13{,}332\]
Example 5
Writing a Sum in Sigma Notation
Express in summation notation: \(\displaystyle \frac{1}{4} + \frac{2}{9} + \frac{3}{14} + \frac{4}{19} + \frac{5}{24} + \frac{6}{29}\)
Find the Numerator Pattern
\[1,\ 2,\ 3,\ 4,\ 5,\ 6\]
Numerators: just \(k\)
Find the Denominator Pattern
\[4,\ 9,\ 14,\ 19,\ 24,\ 29\]
Arithmetic with \(d = 5\) and first term \(4\): each is \(5k - 1\)
Verify
  • \(k=1\): \(\frac{1}{5(1)-1} = \frac{1}{4}\) ✔
  • \(k=2\): \(\frac{2}{5(2)-1} = \frac{2}{9}\) ✔
  • \(k=6\): \(\frac{6}{5(6)-1} = \frac{6}{29}\) ✔
Answer
\[\sum_{k=1}^{6} \frac{k}{5k-1}\]
Your Turn
Practice: Write a Sum in Sigma Notation
Express in summation notation: \(\displaystyle \frac{3}{2} + \frac{5}{6} + \frac{7}{10} + \frac{9}{14}\)
Hint
The numerators \(3, 5, 7, 9\) form one arithmetic sequence, and the denominators \(2, 6, 10, 14\) form another. Write each pattern in terms of \(k\), then choose the correct start and stop values.
Answer
\[\sum_{k=1}^{4}\frac{2k+1}{4k-2}\]
Example 6
Arithmetic-Series Application
A business sells \(\$20{,}000\) during its first year and increases sales by \(\$15{,}000\) each year for 19 more years. Find the total sales over the first 20 years.
Model the Annual Sales
  • \(a_1 = 20{,}000\)
  • \(d = 15{,}000\)
  • \(n = 20\)
Find the 20th Year
\[a_{20} = 20{,}000 + 19(15{,}000) = 305{,}000\]
Add the First 20 Terms
\[S_{20} = \dfrac{20}{2}(20{,}000 + 305{,}000) = 10(325{,}000) = 3{,}250{,}000\]
Answer
\[\$3{,}250{,}000\]
Interpret
Year 1 sales: \(\$20{,}000\). By year 20: \(\$305{,}000\). Cumulative revenue over all 20 years: $3.25 million.
