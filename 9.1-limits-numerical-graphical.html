9.1 Finding Limits: Numerical & Graphical
Objective: Understand limit notation, find limits from graphs and tables, and distinguish limits from function values.
limit notation one-sided limits numerical tables limit vs. value
Reference
Limit Notation
Definition
\[\lim_{x \to a} f(x) = L\]
  • Read: "the limit of \(f(x)\) as \(x\) approaches ______\(a\) equals ______\(L\)."
  • Key point: The limit describes behavior ________near \(a\), not at \(a\). The value \(f(a)\) need not exist, and even if it does, it need not equal \(L\).
Example 1
Identifying Parts of Limit Notation
Identify \(a\), \(f(x)\), and \(L\) in the statement \(\displaystyle\lim_{x \to 2}(3x + 5) = 11\). OpenStax
Identify each piece
\[a = 2, \qquad f(x) = 3x + 5, \qquad L = 11\]
As \(x\) gets close to 2, the output \(3x + 5\) gets close to 11.
Reference
One-Sided & Two-Sided Limits
Left-hand
\[\lim_{x \to a^-} f(x) = L\]
Right-hand
\[\lim_{x \to a^+} f(x) = L\]
  • Left-hand: \(x\) approaches \(a\) from the ________left (\(x < a\)).
  • Right-hand: \(x\) approaches \(a\) from the ________right (\(x > a\)).
  • The two-sided limit exists only when left = right. If they differ, the limit ________________________does not exist (DNE).
Two-Sided Limit Test
\(\displaystyle\lim_{x \to a} f(x) = L \;\iff\; \lim_{x \to a^-} f(x) = L \;\text{and}\; \lim_{x \to a^+} f(x) = L\)
Example 2
One-Sided Limits from a Graph (DNE)
A piecewise function's graph approaches \(y = 8\) from the left at \(x = 2\), and \(y = 3\) from the right. There is a filled dot at \((2, 3)\). Find all four quantities. OpenStax
Read the graph
\[\lim_{x \to 2^-} f(x) = 8, \quad \lim_{x \to 2^+} f(x) = 3, \quad \lim_{x \to 2} f(x) = \textbf{DNE}, \quad f(2) = 3\]
Left \(\ne\) Right, so the two-sided limit does not exist. But the function value \(f(2) = 3\) does exist.
Example 3
Limit Exists but \(\ne f(a)\)
A graph where both one-sided limits at \(x = 2\) equal 8, but \(f(2) = 4\) (filled dot below). Find the limit and \(f(2)\). Are they equal? OpenStax
Analyze
\[\lim_{x \to 2} f(x) = 8 \qquad \text{but} \qquad f(2) = 4\]
The limit exists and equals 8, but \(f(2) \ne \lim_{x \to 2} f(x)\). They are independent.
Concept

Limit vs. Function Value

Independent Quantities
The limit \(\displaystyle\lim_{x \to a} f(x)\) and the function value \(f(a)\) are independent:
  • The limit may exist even if \(f(a)\) is undefined.
  • The limit may differ from \(f(a)\) even if both exist.
  • They agree only when \(f\) is continuous at \(a\) (Section 9.3).
Concept

Finding a Limit Numerically (Table Method)

1. Choose Values
Pick \(x\)-values approaching \(a\) from both sides (e.g. \(a \pm 1, a \pm 0.5, a \pm 0.1, a \pm 0.01\)).
2. Compute \(f(x)\)
Evaluate the function at each value. Record results in a table.
3. Observe
If both sides approach the same \(L\), then \(\lim_{x \to a} f(x) = L\). The table gives evidence, not proof.
Example 4
Limit from a Table
Use a table to estimate \(\displaystyle\lim_{x \to 0}\dfrac{5\sin x}{3x}\). OpenStax
Build the table
\(x\)\(-1\)\(-0.5\)\(-0.1\)\(-0.01\)\(0.01\)\(0.1\)\(0.5\)\(1\)
\(\frac{5\sin x}{3x}\) 1.4021.5981.6641.6671.6671.6641.5981.402
Conclusion
\[\lim_{x \to 0}\frac{5\sin x}{3x} = \boxed{\frac{5}{3}}\]
\(f(0)\) is undefined (division by zero), but the limit exists. Uses the fact that \(\lim_{x\to 0}\frac{\sin x}{x} = 1\).
Example 5
When a Limit Does Not Exist (Oscillation)
Investigate \(\displaystyle\lim_{x \to 0} 3\sin\!\left(\frac{\pi}{x}\right)\). Does the limit exist? OpenStax
Analysis
As \(x \to 0\), the argument \(\pi/x \to \pm\infty\), causing \(\sin(\pi/x)\) to oscillate between \(-1\) and \(1\) with increasing frequency.
The function never settles to a single value.

The limit does not exist.
