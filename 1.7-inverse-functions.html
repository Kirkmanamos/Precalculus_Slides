Chapter 1 · Functions
1.7 Inverse Functions
Objective: Verify inverse functions using composition, find inverse functions algebraically, identify one-to-one functions using the horizontal line test, and restrict domains to find inverses.
one-to-one horizontal line test algebraic inverses composition verification restricted domains
Concept
Inverse Functions Swap Domain & Range
OpenStax
Notation
The inverse of \(f(x)\) is written \(f^{-1}(x)\). Note: this is **not** an exponent, i.e., \(f^{-1}(x) \neq \tfrac{1}{f(x)}\).
Swapping coordinates
If the point \((a,\, b)\) lies on the graph of \(f\), then the point ________\((b,\, a)\) lies on the graph of \(f^{-1}\).
Method
Find an Inverse Algebraically
OpenStax
Step 1
Replace \(f(x)\) with \(y\).
Step 2
Interchange \(x\) and \(y\).
Step 3
Solve the resulting equation for \(y\).
Step 4
Replace \(y\) with \(f^{-1}(x)\). State any domain restrictions on the inverse.
Example 1
Find the inverse of \(f(x) = x^3 - 4\)
Larson
Step 1 & 2 · Replace and Swap
\[x = y^3 - 4\]
Step 3 · Solve for \(y\)
\[x + 4 = y^3 \;\Longrightarrow\; y = \sqrt[3]{x + 4}\]
Step 4 · Rewrite in inverse notation
\[f^{-1}(x) = \sqrt[3]{x + 4}\]
Quick Verification check
\[f(f^{-1}(x)) = \left(\sqrt[3]{x+4}\right)^3 - 4 = (x + 4) - 4 = x \quad\checkmark\]
Example 2
Find the inverse of \(f(x) = \dfrac{3x + 1}{x - 2}\)
OpenStax
Step 1 & 2 · Swap variables
\[x = \dfrac{3y + 1}{y - 2}\]
Step 3 · Multiply by denominator and distribute
\[x(y - 2) = 3y + 1 \;\Longrightarrow\; xy - 2x = 3y + 1\]
Group all \(y\) terms on one side
\[xy - 3y = 2x + 1\]
Factor out \(y\) and solve
\[y(x - 3) = 2x + 1 \;\Longrightarrow\; y = \dfrac{2x + 1}{x - 3}\]
Final Answer
\[f^{-1}(x) = \dfrac{2x + 1}{x - 3}, \quad x \neq 3\]
Example 3
Verify Inverses via Composition
OpenStax
Show that \(f(x) = \dfrac{4}{x - 3}\) and \(g(x) = \dfrac{4}{x} + 3\) are inverses of each other by verifying that \(f(g(x)) = x\) and \(g(f(x)) = x\).
Step 1 · Show \(f(g(x)) = x\)
\[f(g(x)) = f\left(\dfrac{4}{x} + 3\right) = \dfrac{4}{\left(\dfrac{4}{x} + 3\right) - 3} = \dfrac{4}{\dfrac{4}{x}} = 4 \cdot \dfrac{x}{4} = x \quad\checkmark\]
Step 2 · Show \(g(f(x)) = x\)
\[g(f(x)) = g\left(\dfrac{4}{x - 3}\right) = \dfrac{4}{\dfrac{4}{x - 3}} + 3 = (x - 3) + 3 = x \quad\checkmark\]
Conclusion
Since both composite configurations yield \(x\), they are mathematically verified inverse functions.
Concept
Graphical Relationship of Inverses
OpenStax
Reflection Across \(y = x\)
Geometry
The graph of \(f^{-1}(x)\) is the reflection of \(f(x)\) over the line \(y = x\).
Folding Principle
If you fold the coordinate plane along the line \(y = x\), the graph of the function will land exactly on the graph of its inverse.
Concept
Horizontal Line Test & One-to-One
OpenStax
One-to-One Definition
A function is **one-to-one** if each output in the range corresponds to ________________exactly one input in the domain. No two distinct inputs share the same output.
Horizontal Line Test (HLT)
A function has a function inverse if and only if its graph intersects every horizontal line in ________________at most one point.
Core Connection
A function \(f\) has an inverse function \(f^{-1}\) **if and only if** \(f\) is one-to-one. If it fails the horizontal line test, its inverse is a relation, not a function!
Example 4
Is the Function One-to-One?
OpenStax
a) \(f(x) = x^2 - 3\)
NO — fails HLT
b) \(g(x) = x^3\)
YES — passes HLT
c) \(h(x) = \sqrt{x}\)
YES — passes HLT
Concept
Restricting Domain to Force Invertibility
OpenStax
Restricted Domains
Functions that are not one-to-one (like quadratic parabolas) do not have function inverses. However, we can **restrict the domain** of the original function to an interval where it is strictly increasing or decreasing (and thus one-to-one) so that we can find an inverse.
Example parent: \(f(x) = x^2\)
• Unrestricted: domain \((-\infty,\, \infty)\) is not one-to-one.
• Restricted: domain \([0,\, \infty)\) is one-to-one, allowing the inverse \(f^{-1}(x) = \sqrt{x}\).
Mathematical Constraint
When squaring or taking square roots during inverse solving, the restricted domain dictates whether you take the positive or negative branch.
Example 5
Find Inverse on a Restricted Domain
OpenStax
The function \(f(x) = (x - 2)^2\) is not one-to-one. Restrict its domain to \([2,\, \infty)\) and find its inverse function.
Step 1 · Swap variables
\[x = (y - 2)^2 \quad \text{for} \quad y \geq 2\]
Step 2 · Take square root of both sides
Since \(y \geq 2 \;\Longrightarrow\; y - 2 \geq 0\). Therefore, we take the positive root:
\[\sqrt{x} = y - 2\]
Step 3 · Solve for \(y\) and identify domain of the inverse
\[y = \sqrt{x} + 2\] The domain of \(f^{-1}\) is the range of restricted \(f\), which is \([0,\, \infty)\).
Final Answer
\[f^{-1}(x) = \sqrt{x} + 2, \quad x \geq 0\]
Summary
Big Ideas · 1.7
Inverse Function
Swaps inputs and outputs: \(f(a)=b \iff f^{-1}(b)=a\). Domain and range trade places.
Algebraic Finding
Swap \(x\) and \(y\) in the equation, then solve for \(y\). Rewrite with \(f^{-1}(x)\) notation.
Composition Check
To verify, check that both \(f(f^{-1}(x)) = x\) and \(f^{-1}(f(x)) = x\).
Graph Symmetry
Graphs are symmetric reflections across the diagonal identity line \(y = x\).
One-to-One & HLT
Inverse functions exist only if the function is one-to-one (passes Horizontal Line Test).
Restricted Domain
Restrict domains of non-one-to-one functions to force one-to-one behavior and define an inverse.
