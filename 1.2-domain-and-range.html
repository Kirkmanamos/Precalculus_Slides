1.2 Domain and Range
Objective: Find the domain and range of functions algebraically and from graphs, write intervals in interval and set-builder notation, and evaluate/graph piecewise-defined functions.
domain & range interval notation set-builder notation piecewise functions domain restrictions
Definition
What are Domain and Range?
OpenStax
Domain
The domain of a function is the set of all possible ________________input values (typically \(x\)) for which the function is defined and produces a real number output.
Range
The range of a function is the set of all possible ________________output values (typically \(y\)) that the function can produce.
Real-World Constraints
Even if an equation is mathematically defined for all numbers, the context of an application (such as distance, time, or cost) may restrict the domain and range to non-negative values or specific intervals.
Reference
Interval and Set-Builder Notation
OpenStax
Description Inequality Notation Set-Builder Notation Interval Notation
All real numbers \(-\infty < x < \infty\) \(\{x \mid x \in \mathbb{R}\}\) \((-\infty,\, \infty)\)
Numbers greater than or equal to \(a\) \(x \geq a\) \(\{x \mid x \geq a\}\) \([a,\, \infty)\)
Numbers strictly between \(a\) and \(b\) \(a < x < b\) \(\{x \mid a < x < b\}\) \((a,\, b)\)
Closed interval between \(a\) and \(b\) \(a \leq x \leq b\) \(\{x \mid a \leq x \leq b\}\) \([a,\, b]\)
Parentheses \(()\)
Used when an endpoint is ________________excluded (strictly less/greater than, or infinity).
Brackets \([\,]\)
Used when an endpoint is ________________included (less/greater than or equal to).
Quick Check
Bracket or parenthesis?
Which interval notation represents \(x \le 7\)?
Correct. \(x \le 7\) includes everything up to and including 7 — bracket on 7, and \(\infty\) always gets a parenthesis.
\(x \le 7\) means all values up to 7. It goes down to \(-\infty\), and the \(\le\) includes 7 — so a bracket: \((-\infty,\, 7]\).
Concept
Algebraic Domain Restrictions
OpenStax
1. Division by Zero
Denominator cannot equal zero.
If \(f(x) = \dfrac{g(x)}{h(x)}\), set ________________\(h(x) \neq 0\).
Exclude any values of \(x\) that make \(h(x) = 0\).
2. Even Roots of Negatives
Radicand of an even index root must be non-negative.
If \(f(x) = \sqrt[2k]{g(x)}\), set ________________\(g(x) \geq 0\).
Solve the inequality for allowed \(x\) values.
Pro Tip
Polynomials (like \(y=x^2-3x+2\)) and odd roots (like \(y=\sqrt[3]{x}\)) have no algebraic restrictions. Their domain is all real numbers, \((-\infty,\, \infty)\).
Example 1
Find the Domain Algebraically
OpenStax
Find the domain of each function in interval notation.
a) \(f(x) = \dfrac{x+1}{2-x}\)
Set denominator \(\neq 0\): \(\;2 - x \neq 0 \;\Longrightarrow\; x \neq 2\)
Domain: \((-\infty,\, 2) \cup (2,\, \infty)\)
b) \(g(x) = \sqrt{7-x}\)
Set radicand \(\geq 0\): \(\;7 - x \geq 0 \;\Longrightarrow\; 7 \geq x \;\Longrightarrow\; x \leq 7\)
Domain: \((-\infty,\, 7]\)
c) \(h(x) = \dfrac{5}{\sqrt{x-3}}\)
Combined restriction: Radicand must be \(\geq 0\) AND denominator \(\neq 0\).
Set radicand strictly positive: \(\;x - 3 > 0 \;\Longrightarrow\; x > 3\)
Domain: \((3,\, \infty)\)
Summary Rule
Denominator \(\neq 0 \;\Longrightarrow\; \text{exclude endpoints. } \sqrt{u} \text{ in denominator } \Longrightarrow\; u > 0.\)
Quick Check
Two restrictions at once
What is the domain of \(h(x) = \dfrac{5}{\sqrt{x - 3}}\)?
Correct. The root needs \(x - 3 \ge 0\), but it’s in the denominator too, so \(x - 3 \ne 0\). Together: \(x - 3 > 0\), i.e. \((3,\infty)\).
Two rules stack here: \(\sqrt{x-3}\) needs \(x-3 \ge 0\), and as a denominator it can’t be 0. So \(x - 3 > 0\) — strictly, giving \((3,\infty)\).
Concept
Domain and Range from Graphs
OpenStax
Scan horizontally for Domain
Look left-to-right along the \(x\)-axis.
• Left boundary is the start of the inputs.
• Right boundary is the end of the inputs.
Scan vertically for Range
Look bottom-to-top along the \(y\)-axis.
• Lower boundary is the lowest value reached.
• Upper boundary is the highest value reached.
Closed circle \((\bullet)\)
Indicates the value is ________________included in the set. Uses brackets \([\,]\).
Open circle \((\circ)\)
Indicates the value is ________________excluded from the set. Uses parentheses \(()\).
Example 2
Identify Domain and Range from Graph
Larson
State the domain and range of the function shown in the graph.
Domain (x-axis scan)
Starts at \(x = -3\) (open circle)
Ends at \(x = 5\) (closed circle)
Domain: \((-3,\, 5]\)
Range (y-axis scan)
Lowest point is at \(y = -2\) (included, solid valley)
Highest point is at \(y = 4\) (excluded, open circle)
Range: \([-2,\, 4)\)
Key Takeaway
Do not just look at the endpoints for the range! Check local peaks and valleys as they define the absolute boundaries.
Quick Check
Where’s the range?
To find the range of a function from its graph, you should…
Correct. The range is the full vertical span. A valley or peak between the endpoints can set the true minimum or maximum.
Endpoints aren’t enough — a valley or peak between them can be the real min/max. Scan the whole vertical span. (Scanning left-to-right gives the domain.)
Concept
What is a Piecewise Function?
OpenStax
Definition
A piecewise function is a function defined by more than one formula, where each formula applies to a different part of the domain.
\[f(x) = \begin{cases} x + 1, & \text{if } x < -2 \\ -2x - 3, & \text{if } x \geq -2 \end{cases}\]
1. How to Evaluate
Check the input value against the domain conditions. Plug it into only the equation that corresponds to that interval.
2. Finding the Overall Domain
Combine all individual pieces' sub-domains: \((-\infty,\, -2) \cup [-2,\, \infty) = (-\infty,\, \infty)\).
Example 3
Analyze the Piecewise Function
OpenStax
For the piecewise function \(f(x)\) from Slide 7, evaluate at coordinates and analyze the graph.
Step 1 · Evaluate
• \(f(-3)\): since \(-3 < -2\), use piece 1:
\(f(-3) = -3 + 1 = -2\)
• \(f(-2)\): since \(-2 \geq -2\), use piece 2:
\(f(-2) = -2(-2) - 3 = 1\)
Step 4 · Identify the Range
• Piece 1 heads to \(-\infty\) (starting from open \(y = -1\)).
• Piece 2 heads to \(-\infty\) (starting from closed \(y = 1\)).
Highest output reached overall is \(y = 1\).
Range: \((-\infty,\, 1]\)
Example 4
Applied Domain and Range
OpenStax
A cellular provider charges \(\$30\) per month for a basic plan, plus \(\$10\) per GB of data used, capped at a maximum of \(5\) GB. Write the cost function \(C(g) = 30 + 10g\), and find the domain and range in this context.
Step 1 · Find the Domain (data usage, g)
Data usage cannot be negative, and is capped at \(5\) GB.
Domain: \([0,\, 5]\)
Step 2 · Find the Range (total cost, C)
Evaluate the cost function at the domain boundaries:
• Minimum cost: \(C(0) = 30 + 10(0) = \$30\)
• Maximum cost: \(C(5) = 30 + 10(5) = \$80\)
Range: \([30,\, 80]\)
Real-World vs. Math limits
Mathematically, \(y = 30 + 10x\) has domain \((-\infty,\, \infty)\). However, physical boundaries (negative data is impossible, and the carrier caps use) dictate the practical domain.
Summary
Big Ideas · 1.2
Domain vs Range
Domain is the set of all allowed inputs (\(x\)); Range is the set of all outputs (\(y\)).
Notation
Use \([\,]\) brackets for included values and \(()\) parentheses for excluded values and infinity.
Restrictions
Algebraic domain excludes values causing division by zero or even roots of negatives.
Graphs
Domain is horizontal span (left to right). Range is vertical span (bottom to top). Watch for local extrema.
Piecewise functions
Formulas apply to different intervals. Evaluate by choosing the correct piece; graph step-by-step.
Context
Applied problems restrict variables to physical bounds, rendering standard algebraic domains obsolete.
