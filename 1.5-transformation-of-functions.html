1.5 Transformation of Functions
Objective: Identify translations, reflections, and dilations of parent functions from their equations, use transformations to graph functions, and classify functions as even, odd, or neither.
parent functions translations reflections stretches & compressions even & odd functions
Concept
Parent Functions You Should Know
OpenStax
Linear
\(f(x)=x\)
Quadratic
\(f(x)=x^2\)
Cubic
\(f(x)=x^3\)
Absolute Value
\(f(x)=|x|\)
Square Root
\(f(x)=\sqrt{x}\)
Cube Root
\(f(x)=\sqrt[3]{x}\)
Why Memorize Them?
Every transformed graph in this section starts from one of these parents. Recognize the parent formula, then apply shifts, reflections, and stretches.
Concept
Three Flavors of Transformation
OpenStax
Translation
\(g(x) = f(x - h) + k\)
Slide the graph horizontally by \(h\) or vertically by \(k\). Size and shape do not change.
Reflection
\(-f(x)\) or \(f(-x)\)
Flip across the \(x\)-axis (negate outside) or the \(y\)-axis (negate inside).
Dilation
\(c \cdot f(x)\) or \(f(k x)\)
Stretch or shrink the graph vertically by \(c\) or horizontally by \(\tfrac{1}{k}\).
Core Principle
Changes outside the function affect \(y\) (vertical direction) and behave as expected. Changes inside the function affect \(x\) (horizontal direction) and behave opposite to how they look.
Reference
Translations · Shifts
OpenStax
Horizontal Shift
\(f(x - h)\) shifts the graph ________________right by \(h\) units when \(h > 0\).
Vertical Shift
\(f(x) + k\) shifts the graph ________________up by \(k\) units when \(k > 0\).
Watch the Sign!
\(f(x - 3)\) means shift right 3 (opposite). \(f(x + 3)\) means shift left 3. Vertically, \(f(x) + 3\) goes up 3 and \(f(x) - 3\) goes down 3 (as expected).
Quick Check
Which way does it shift?
The graph of \(g(x) = f(x + 4)\) is the graph of \(y = f(x)\) shifted…
Correct. The \(+4\) is inside the function, so it acts on \(x\) and behaves opposite to its sign — a shift left 4.
Look inside the parentheses. Inside changes act on \(x\) and behave opposite to the sign, so \(+4\) shifts the graph left.
Reference
Reflections and Dilations
OpenStax
\(-f(x)\) · outside
Reflect over the ________\(x\)-axis. Negates all \(y\)-values.
\(f(-x)\) · inside
Reflect over the ________\(y\)-axis. Negates all \(x\)-values.
\(c \cdot f(x)\) · vertical dilation
\(|c|>1\): stretch away from \(x\)-axis. \(0<|c|<1\): compress toward \(x\)-axis.
\(f(k x)\) · horizontal dilation
\(|k|>1\): compress toward \(y\)-axis by \(\tfrac{1}{k}\). \(0<|k|<1\): stretch away from \(y\)-axis.
Quick Check
Negate the outside
The transformation \(y = -f(x)\) reflects the parent graph over which axis?
Correct. The negative sign is outside, so it negates every \(y\)-value — a reflection over the \(x\)-axis.
The negative is outside the function, so it acts on the \(y\)-values. Negating outputs flips the graph over the \(x\)-axis.

Tip
Order of Transformations — Use D.R.S.
OpenStax
When does order matter?
If a function has multiple transformations in the same direction (horizontal or vertical), the order of application matters. A shift must happen after a reflection or dilation.
D · R · S
D · Dilate
Apply stretches and shrinks first.
R · Reflect
Apply flips over the axes second.
S · Shift
Apply vertical/horizontal translations last.
Mnemonic
Dilate → Reflect → Shift. D.R.S. keeps your points aligned.
Quick Check
Which one happens last?
A graph has a stretch, a reflection, and a shift. Using D.R.S., which is applied last?
Correct. \(\textbf{D} \rightarrow \textbf{R} \rightarrow \textbf{S}\): dilate, reflect, then shift last, so the translation lands on the already-stretched, reflected graph.
Recall the order \(\textbf{D} \rightarrow \textbf{R} \rightarrow \textbf{S}\). The Shift comes last so it acts on the dilated, reflected graph.
Concept
Even and Odd Functions
OpenStax
Even · \(y\)-axis symmetry
\(f(-x) = f(x)\)
Replacing \(x\) with \(-x\) results in the original function. Symmetric over the \(y\)-axis.
Odd · origin symmetry
\(f(-x) = -f(x)\)
Replacing \(x\) with \(-x\) negates all outputs. Symmetric over the origin (\(180^\circ\) rotation).
Algebraic Test Strategy
Substitute \(-x\) for every \(x\) in the function. • If it simplifies back to \(f(x)\), it is ________Even.
• If it simplifies to \(-f(x)\) (all signs flipped), it is ________Odd.
Example 2
Determine if Even, Odd, or Neither
OpenStax
Classify each function algebraically.
a) \(f(x) = x^3 - x\)
\[f(-x) = (-x)^3 - (-x) = -x^3 + x = -(x^3 - x) = -f(x) \quad\Longrightarrow\quad \text{\textbf{Odd}}\]
b) \(g(x) = x^2 + 2\)
\[g(-x) = (-x)^2 + 2 = x^2 + 2 = g(x) \quad\Longrightarrow\quad \text{\textbf{Even}}\]
c) \(h(x) = x^3 - 1\)
\[h(-x) = (-x)^3 - 1 = -x^3 - 1\] Does not equal \(h(x)\) (not even), and does not equal \(-h(x) = -x^3 + 1\) (not odd) \(\;\Longrightarrow\quad \text{\textbf{Neither}}\)
Polynomial Tip
If all exponents are even, the function is even. If all exponents are odd (and no constant term), it is odd. Mixed exponents (like \(x^3-1 = x^3 - x^0\)) make it neither.
Quick Check
Even, odd, or neither?
Classify \(p(x) = x^4 - 3x^2\).
Correct. \(p(-x) = (-x)^4 - 3(-x)^2 = x^4 - 3x^2 = p(x)\). Every exponent is even, so the graph is symmetric over the \(y\)-axis.
Substitute \(-x\): \(p(-x) = x^4 - 3x^2 = p(x)\). Both exponents are even powers, so \(p\) is even.

Summary
Big Ideas · 1.5
Inside vs. Outside
Inside changes affect \(x\) horizontally (behaves opposite). Outside changes affect \(y\) vertically (behaves as expected).
Order of Operations
Apply in D.R.S. order: Dilations (stretches/shrinks), then Reflections (flips), then Shifts (translations).
Even Functions
\(f(-x) = f(x)\). Symmetric over the \(y\)-axis. Even powers of \(x\) or absolute value functions.
Odd Functions
\(f(-x) = -f(x)\). Symmetric over the origin. Odd powers of \(x\) and no constants.
Dilation factor
Vertical stretches have factor \(c > 1\). Horizontal compressions have inside multiplier \(k > 1\) (factor \(\tfrac{1}{k}\)).
Mapping points
To graph easily, write down key points of parent function, then track coordinate modifications step-by-step.
