Chapter 1 · Functions
1.6 Absolute Value Functions
Objective: Graph absolute value functions using transformations, solve absolute value equations algebraically, and solve absolute value inequalities.
piecewise definition vertex \((h,\, k)\) equations inequalities number lines
Definition
The Absolute Value Function
OpenStax
Piecewise Definition
The function \(f(x) = |x|\) yields the distance from \(x\) to origin. It is defined piecewise as: \[f(x) = |x| = \begin{cases} x, & \text{if } x \geq 0 \\ -x, & \text{if } x < 0 \end{cases}\]
Properties
• Domain: \((-\infty,\, \infty)\)
• Range: \([0,\, \infty)\)
• Vertex: The corner point where the direction changes is the ________vertex, which is at \((0,\,0)\) for the parent function.
Concept
Graphing with Transformations
OpenStax
General Form: \(f(x) = a|x - h| + k\)
Rather than plotting arbitrary coordinates, we can graph by identifying the transformations:
• Vertex: Shifted from \((0,\,0)\) to ________\((h,\, k)\).
• Orientation: If \(a > 0\), the V opens up. If \(a < 0\), the V opens down.
• Slopes of Branches: The right side has slope \(a\) and the left side has slope \(-a\).
Quick Graphing Routine
1. Plot the vertex at \((h,\, k)\).
2. Use the slope \(a\) to plot at least one point to the right: \((h+1,\, k+a)\).
3. Reflect that point across the vertical line of symmetry \(x = h\) to get a point on the left: \((h-1,\, k+a)\).
4. Draw the V-shape connecting these points.
Example 1
Graph \(f(x) = -2|x - 3| + 4\)
OpenStax
Step 1 · Identify Vertex \((h, k)\)
Since \(h = 3\) and \(k = 4\):
Vertex: \((3,\, 4)\)
Step 2 · Determine Shape and Slopes
\(a = -2 \;\Longrightarrow\; \text{Opens down}\).
Branch slopes: \(-2\) (right) and \(2\) (left).
Step 3 · Plot reference points
\(x\)\(1\)\(2\)\(3\) (vertex)\(4\)\(5\)

Quick Check
Find the vertex
What is the vertex of \(f(x) = 2|x + 1| - 3\)?
Correct. \(|x + 1| = |x - (-1)|\), so \(h = -1\) and \(k = -3\): vertex \((-1,\, -3)\).
Match the form \(a|x - h| + k\): \(x + 1 = x - (-1)\), so \(h = -1\) (not \(+1\)) and \(k = -3\). Vertex \((-1,\, -3)\).
Concept
Solving Absolute Value Equations
OpenStax
General Strategy
To solve an equation containing an absolute value, first isolate the absolute value term on one side: \[|u| = c\]
Case 1 · If \(c < 0\)
No solutions. Absolute value measures distance and cannot be negative.
Case 2 · If \(c = 0\)
Exactly one equation to solve: \(\;u = 0\).
Case 3 · If \(c > 0\)
Set up two separate equations to solve: \(\;u = c \quad\text{and}\quad u = -c\).
Example 2
Solve the Equation
OpenStax
Solve the equation: \(\;2|3x - 1| + 5 = 15\)
Step 1 · Isolate the absolute value term
\[2|3x - 1| = 10 \;\Longrightarrow\; |3x - 1| = 5\]
Step 2 · Split into two cases (since \(5 > 0\))
\[3x - 1 = 5 \qquad\text{or}\qquad 3x - 1 = -5\]
Step 3 · Solve Case 1
\[3x = 6 \;\Longrightarrow\; x = 2\]
Step 4 · Solve Case 2
\[3x = -4 \;\Longrightarrow\; x = -\dfrac{4}{3}\]
Final Solution Set
\[\left\{2,\, -\dfrac{4}{3}\right\}\]
Quick Check
How many solutions?
How many solutions does \(|2x - 1| + 7 = 3\) have?
Correct. Isolate first: \(|2x - 1| = -4\). An absolute value can never be negative, so there’s no solution.
Always isolate the bars first: \(|2x - 1| = 3 - 7 = -4\). Distance can’t be negative, so no solution — don’t split into cases.
Concept
Solving Absolute Value Inequalities
OpenStax
General Rules (for \(c > 0\))
Translate the absolute value inequality into a compound inequality without absolute value bars.
1. Less-Than ("Less-and")
\[|u| < c \;\Longrightarrow\; -c < u < c\] Yields a single bounded interval. Input is close to the vertex.
2. Greater-Than ("Great-or")
\[|u| > c \;\Longrightarrow\; u < -c \quad\text{or}\quad u > c\] Yields a union of two unbounded intervals. Input is far from the vertex.
Equalities Included
The same rules apply for \(\leq\) and \(\geq\), simply include the endpoints (use square brackets in interval notation).
Example 3
Solve the "Less-Than" Inequality
OpenStax
Solve and graph the solution: \(\;|x - 5| < 4\)
Step 1 · Translate to compound inequality
\[-4 < x - 5 < 4\]
Step 2 · Add 5 to all three parts
\[1 < x < 9\]
Interval Notation
\[(1,\, 9)\]
Example 4
Solve the "Greater-Than" Inequality
OpenStax
Solve and graph the solution: \(\;|2x + 3| \geq 5\)
Step 1 · Split into two separate inequalities
\[2x + 3 \geq 5 \qquad\text{or}\qquad 2x + 3 \leq -5\]
Step 2 · Solve Case 1
\[2x \geq 2 \;\Longrightarrow\; x \geq 1\]
Step 3 · Solve Case 2
\[2x \leq -8 \;\Longrightarrow\; x \leq -4\]
Interval Notation
\[(-\infty,\, -4] \cup [1,\, \infty)\]
Quick Check
“Great-OR” or “Less-AND”?
The solution to \(|x| > 3\) is…
Correct. “Greater-than” means far from 0 — two rays: \(x < -3\) or \(x > 3\).
“Great-OR”: \(|x| > 3\) means far from 0, giving two rays \(x<-3\) or \(x>3\). (B is the answer to \(|x| < 3\).)
Summary
Big Ideas · 1.6
V-Shape Graph
Graph has a sharp corner at the vertex \((h,\, k)\). Use branch slope \(a\) to plot points.
Piecewise split
Parent function is \(|x| = x\) for \(x \geq 0\) and \(-x\) for \(x < 0\). Always non-negative outputs.
Equations
Isolate \(|u|\). If positive, solve \(u = c\) and \(u = -c\). If negative, no solution.
Less-Than ("Less-and")
\(|u| < c \;\Longrightarrow\; -c < u < c\). Yields a single bounded interval.
Greater-Than ("Great-or")
\(|u| > c \;\Longrightarrow\; u < -c \text{ or } u > c\). Yields two disjoint intervals.
Graphing inequalities
Represent solutions on a number line. Use open circles for \(<\) / \(>\) and solid circles for \(\leq\) / \(\geq\).
