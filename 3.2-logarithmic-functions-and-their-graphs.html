3.2 Logarithmic Functions & Their Graphs
Objective: Define the logarithm as the inverse of an exponential, convert between forms, evaluate using basic properties, and graph logarithmic functions (with transformations).
inverse of exp common & natural log basic properties graphs transformations
Definition
Logarithm with Base \(b\)
A logarithm answers the question: “to what power do I raise \(b\) to get \(n\)?” It is the inverse operation of an exponential.
Definition
For \(b > 0,\ b \neq 1\):
\(\log_{b} n = p\) ⇔ \(b^{p} = n\)
Read it as
“Log base \(b\) of \(n\) equals \(p\)” — i.e., \(p\) is the exponent you put on \(b\) to get \(n\). A log is an exponent.

Restrictions
  • Base \(b > 0\) and \(b \neq 1\).
  • Argument must be positive: you can take \(\log\) of a positive number only.
  • Output \(p\) can be any real number.
Example 1
Conversion between log and exponential form
(a) Write \(\log_{m} p = q\) in exponential form.  (b) Write \(z^{y} = w\) in logarithmic form.
(a) log → exponential
\[\log_{m} p = q \quad\Longleftrightarrow\quad m^{q} = p\]
(b) exponential → log
\[z^{y} = w \quad\Longleftrightarrow\quad \log_{z} w = y\]
Definitions
Common Log and Natural Log
Common Log
  • The log function with base 10.
  • When the base is 10, it is not necessary to write the base.
\(\log x = \log_{10} x\)
Natural Log
  • The log function with base \(e\).
  • The natural log is written \(\ln x\).
\(\ln x = \log_{e} x\)
Natural Log Definition
\(\ln m = p \quad\Longleftrightarrow\quad e^{p} = m\)

Why these work
The last two are pure “inverse cancellation”: \(\log\) and the matching exponential undo each other. The “one-to-one” rule lets you drop matching logs on both sides of an equation.
Example 2
Simplify
(a) \(\log_{2} 2^{5}\)   (b) \(7^{\log_{7} 14}\)   (c) \(\log 10\)   (d) Solve \(\log_{2} x = \log_{2} 3\)
(a) inverse cancellation
\[\log_{2} 2^{5} = 5\]
(b) the other inverse cancellation
\[7^{\log_{7} 14} = 14\]
(c) \(\log a = 1\) when \(a\) is the base
\[\log 10 = \log_{10} 10 = 1\]
(d) one-to-one property
\[\log_{2} x = \log_{2} 3 \ \Longrightarrow\ x = 3\]
Example 3
Simplify natural logs
(a) \(\ln \tfrac{1}{e}\)   (b) \(e^{\ln 5}\)   (c) \(4 \ln 1\)   (d) \(2 \ln e\)
(a) rewrite \(\tfrac{1}{e}\) as \(e^{-1}\)
\[\ln \tfrac{1}{e} = \ln e^{-1} = -1\]
(b) inverse cancellation
\[e^{\ln 5} = 5\]
(c) \(\ln 1 = 0\)
\[4 \ln 1 = 4(0) = 0\]
(d) \(\ln e = 1\)
\[2 \ln e = 2(1) = 2\]
Example 4
Find the inverse of \(g(x) = -2 + \log_{3}(x + 1)\)
Use the swap-and-solve strategy: rewrite \(y = g(x)\), swap \(x\) and \(y\), then solve for \(y\).
Rewrite
\[y = -2 + \log_{3}(x + 1)\]
Swap \(x\) and \(y\)
\[x = -2 + \log_{3}(y + 1)\]
Isolate the log
\[x + 2 = \log_{3}(y + 1)\]
Convert to exponential form
\[3^{\,x + 2} = y + 1\]
Solve for \(y\) (the inverse)
\[g^{-1}(x) = 3^{\,x + 2} - 1\]

Inverse swap
Notice every feature swaps \(x \leftrightarrow y\). Domain becomes range. Horizontal asymptote becomes vertical. \((0, 1)\) becomes \((1, 0)\). That’s the geometric meaning of inverse.
Reference
Transformations of Log Functions & Graphing Strategy
\[f(x) = a \cdot \log_{b}(x - h) + k\]
\(h\) — Horizontal Translation
Shifts right by \(h\). The vertical asymptote moves to \(x = h\).
\(k\) — Vertical Translation
Shifts up by \(k\) (down if negative). The VA is unchanged.
\(a < 0\)
Reflection across the \(x\)-axis.
\(|a| > 1\)
Vertical stretch by factor \(|a|\).
\(0 < |a| < 1\)
Vertical compression by factor \(|a|\).
How to Graph a Log Function
  1. Identify and draw the vertical asymptote \(x = h\).
  2. Identify the transformations from \(a, h, k\).
  3. Write the parent’s exponential inverse (e.g., parent \(\log_{2} x\) ⇒ inverse \(y = 2^{x}\)).
  4. Make a t-table for the inverse using \(x = -1, 0, 1\).
  5. Swap the t-table columns to get the parent log’s points.
  6. Adjust each \((x, y)\) for the transformations.
  7. Plot the points and sketch the curve.
