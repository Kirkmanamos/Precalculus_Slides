Unit 1 · Function Analysis
1.2 Non-Linear Inequalities
Objective: Solve absolute value, quadratic, and rational inequalities using sign diagrams and compound inequalities.
absolute value quadratic rational sign diagrams interval notation
Concept
Inequalities
An inequality is a statement that two quantities or expressions are not equal. Example: \(2x + 3 > 11\).
Inequality symbols
\(<\)   less than
\(>\)   greater than
\(\le\)   less than or equal to
\(\ge\)   greater than or equal to
What it means to "solve"
To solve an inequality means to find all __________values of \(x\) that make the inequality ______true.
Why non-linear?
Unlike linear inequalities, absolute value, quadratic, and rational inequalities can flip direction — solutions often come in unions of intervals, not single intervals.
Strategy
How to Solve an Absolute Value Inequality
Case 1 · \(|mx+b| < a\)
1. Write an AND compound inequality: \[-a < mx + b < a\] 2. Solve for \(x\).
3. Check your answer.
Case 2 · \(|mx+b| > a\)
1. Write an OR compound inequality: \[mx+b > a \ \text{ or }\ mx+b < -a\] 2. Solve for \(x\).
3. Check your answer.
Memory trick
"Less thAND, greatOR." Less-than gives AND (a sandwich). Greater-than gives OR (two pieces).
Always isolate first
Before you split into cases, the absolute value must be alone on one side. Multiply, divide, add, subtract — get \(|mx+b|\) by itself first.
Example 1
Solve \(|x - 3| \le 5\)
This is Case 1 (less than) — write an AND compound inequality.
Step 1 · Write compound inequality
\[-5 \le x - 3 \le 5\]
Case 1 template: \(-a \le mx+b \le a\).
Step 2 · Add 3 to all three parts
\[-2 \le x \le 8\]
Solution · Interval notation
\[x \in [-2,\, 8]\]
Number line
Example 2
Solve \(3|2x + 3| + 1 > 28\)
Isolate the absolute value first, then apply Case 2 (greater than).
Step 1 · Subtract 1, divide by 3
\[|2x + 3| > 9\]
Isolate the absolute value before splitting cases.
Step 2 · OR compound inequality
\[2x+3 > 9 \quad\text{or}\quad 2x+3 < -9\]
Step 3 · Solve each branch
\[2x > 6 \quad\text{or}\quad 2x < -12\]
Solution
\[x > 3 \quad\text{or}\quad x < -6\]
Interval: \((-\infty,\,-6) \cup (3,\,\infty)\).
Number line
Strategy
How to Solve a Quadratic Inequality
  1. Rewrite the inequality with zero on one side and the quadratic on the other.
  2. Find the zeros by solving \(ax^2 + bx + c = 0\). These are the critical values.
  3. Use the zeros to divide a number line into intervals — this is the sign diagram.
  4. Test a value from each interval to see where the inequality holds true.
Why this works
A polynomial can only change sign at its zeros. Between two consecutive zeros, the sign is constant — so testing one point per interval tells you the sign of the entire interval.
Example 3
Solve \(x^2 - 3x - 10 > 0\)
Already in standard form. Factor, then build a sign diagram.
Step 1 · Factor the quadratic
\[(x - 5)(x + 2) > 0\]
Step 2 · Critical values
\[x = -2,\ \ x = 5\]
These are the zeros where the expression can change sign.
Step 3 · Sign diagram · Test each interval
Test \(x=-3\):
\((-3-5)(-3+2) = (-8)(-1) = \) +8 > 0 ✓
Test \(x=0\):
\((0-5)(0+2) = (-5)(2) = \) -10 < 0 ✗
Test \(x=6\):
\((6-5)(6+2) = (1)(8) = \) +8 > 0 ✓
Solution
\[x < -2 \quad\text{or}\quad x > 5 \ \Rightarrow\ (-\infty,\,-2) \cup (5,\,\infty)\]
Strict inequality (\(>\)), so endpoints are excluded.
Number line
Example 4
Solve \(2x^2 - x \le 3\)
Move everything to one side, factor, then build a sign diagram.
Step 1 · Set ≤ 0
\[2x^2 - x - 3 \le 0\]
Subtract 3 from both sides first.
Step 2 · Factor
\[(2x - 3)(x + 1) \le 0\]
Step 3 · Critical values
\[x = -1,\ \ x = \tfrac{3}{2}\]
Step 4 · Sign diagram
Test \(x=-2\):
\((2(-2)-3)(-2+1) = (-7)(-1) = \) +7 ✗
Test \(x=0\):
\((2(0)-3)(0+1) = (-3)(1) = \) -3 ✓
Test \(x=2\):
\((2(2)-3)(2+1) = (1)(3) = \) +3 ✗
Solution
\[-1 \le x \le \tfrac{3}{2} \ \Rightarrow\ \left[-1,\, \tfrac{3}{2}\right]\]
Non-strict inequality (\(\le\)), so endpoints are included.
Strategy
How to Solve a Rational Inequality
  1. Rewrite with zero on one side: \(\dfrac{P(x)}{Q(x)} < 0\) or \(\dfrac{P(x)}{Q(x)} > 0\).
  2. Find the zeros of both the numerator and denominator — all are critical values.
  3. Use the critical values to divide a number line into intervals (sign diagram).
  4. Test a value from each interval.
  5. Exclude values that make the denominator zero — always.
Don't just multiply by the denominator
You don't know the sign of \(Q(x)\), so multiplying both sides could flip the inequality — invalid. Always move everything to one side first, then combine into a single fraction.
Example 5
Solve \(\dfrac{x+2}{x-3} \le 0\)
Already has zero on one side. Identify critical values and test.
Step 1 · Critical values
\[x = -2\ \text{(numerator zero)},\ \ x = 3\ \text{(excluded)}\]
\(x = 3\) makes the denominator zero — never a solution.
Step 2 · Test each interval
\(x=-3\):
\(\dfrac{-1}{-6} = \) + ✗
\(x=0\):
\(\dfrac{2}{-3} = \) − ✓
\(x=4\):
\(\dfrac{6}{1} = \) + ✗
Solution
\[-2 \le x < 3 \ \Rightarrow\ [-2,\, 3)\]
Include \(-2\) (inequality is \(\le\)); exclude \(3\) (undefined).
Number line
Example 6
Solve \(\dfrac{x^2 + x}{x^2 - 1} \ge 0\)
Factor first — common factors matter here.
Step 1 · Factor
\[\dfrac{x(x+1)}{(x-1)(x+1)} \ge 0\]
Numerator: \(x^2+x = x(x+1)\). Denominator: \(x^2-1 = (x-1)(x+1)\).
Step 2 · Critical values
\[x = -1,\ 0,\ 1\]
\(x = -1\) and \(x = 1\) are excluded — they make the denominator zero.
Step 3 · Test each interval
\(x=-2\):
\(\dfrac{(-2)(-1)}{(-3)(-1)} = \dfrac{2}{3}\) + ✓
\(x=-\tfrac{1}{2}\):
\(\dfrac{(-\tfrac12)(\tfrac12)}{(-\tfrac32)(\tfrac12)} = \dfrac{-1/4}{-3/4}\) + ✓
\(x=\tfrac{1}{2}\):
\(\dfrac{(\tfrac12)(\tfrac32)}{(-\tfrac12)(\tfrac32)}\) − ✗
\(x=2\):
\(\dfrac{(2)(3)}{(1)(3)} = 2\) + ✓
Include x = 0?
\[\text{Yes — numerator is zero, denominator is } -1 \ne 0.\]
\(\dfrac{0}{-1} = 0\), which satisfies \(\ge 0\).
Solution
\[(-\infty,\,-1) \cup (-1,\, 0] \cup (1,\,\infty)\]
Exclude \(x=\pm 1\); include \(x=0\).
Example 7
Solve \(\dfrac{x+1}{x+3} < 2\)
Need zero on one side — subtract 2 and combine.
Step 1 · Move 2 across
\[\dfrac{x+1}{x+3} - 2 < 0\]
Step 2 · Common denominator
\[\dfrac{(x+1) - 2(x+3)}{x+3} < 0 \ \Rightarrow\ \dfrac{-x - 5}{x+3} < 0\]
Step 3 · Multiply by \(-1\), flip sign
\[\dfrac{x+5}{x+3} > 0\]
Flipping the inequality makes the numerator cleaner.
Step 4 · Sign diagram
\(x=-6\):
\(\dfrac{-1}{-3}\) + ✓
\(x=-4\):
\(\dfrac{1}{-1}\) − ✗
\(x=0\):
\(\dfrac{5}{3}\) + ✓
Solution
\[x < -5 \ \text{or}\ x > -3 \ \Rightarrow\ (-\infty,\,-5) \cup (-3,\,\infty)\]
Example 8
Solve \(\dfrac{x-1}{x+2} > \dfrac{x-3}{x-4}\)
Two rational expressions — move everything to one side, combine over a single denominator.
Step 1 · Move and combine
\[\dfrac{x-1}{x+2} - \dfrac{x-3}{x-4} > 0\]
Step 2 · Common denominator
\[\dfrac{(x-1)(x-4) - (x-3)(x+2)}{(x+2)(x-4)} > 0\]
Step 3 · Expand numerator
\[\dfrac{(x^2 - 5x + 4) - (x^2 - x - 6)}{(x+2)(x-4)} > 0\]
Step 4 · Simplify
\[\dfrac{-4x + 10}{(x+2)(x-4)} > 0 \ \Rightarrow\ \dfrac{2x - 5}{(x+2)(x-4)} < 0\]
Factor out \(-2\) and flip the inequality.
Step 5 · Critical values & sign test
\(x=-3\):
\(\dfrac{-11}{(-1)(-7)}\) − ✓
\(x=0\):
\(\dfrac{-5}{(2)(-4)}\) + ✗
\(x=3\):
\(\dfrac{1}{(5)(-1)}\) − ✓
\(x=5\):
\(\dfrac{5}{(7)(1)}\) + ✗
Solution
\[(-\infty,\,-2) \cup \left(\tfrac{5}{2},\, 4\right)\]
Critical values: \(-2,\ \tfrac{5}{2},\ 4\). Exclude \(-2\) and \(4\) (denominator zeros).
Summary
Big Ideas · 1.2
Absolute value
Isolate first. Less-than → AND (sandwich). Greater-than → OR (two rays).
Quadratic
Move to zero, factor to find critical values, test each interval on the sign diagram.
Rational
Never multiply by the denominator — combine into one fraction instead. Exclude denominator zeros.
Sign diagrams work
Because expressions only change sign at zeros (or undefined points), one test value per interval reveals the sign of the whole interval.
Endpoint rule
Strict (\(<,\,>\)) → open circle. Non-strict (\(\le,\,\ge\)) → closed circle, unless it makes the denominator zero.
Interval notation
Use \(\cup\) to join disjoint pieces. Parentheses for excluded endpoints, brackets for included.
