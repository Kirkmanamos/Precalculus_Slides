8.5 Parametric Graphs
Objective: Graph parametric curves with direction arrows and apply parametric equations to projectile motion.
t-table graphing orientation arrows projectile motion
Concept

Graphing a Parametric Curve

1. Build T-Table
Create columns for \(t\), \(x(t)\), \(y(t)\). Evaluate at evenly-spaced values across the given interval.
2. Plot & Connect
Plot each \((x, y)\) pair and connect points with a smooth curve.
3. Add Arrows
Draw ____________________directional arrows showing the path as \(t\) increases.
Example 1
Graphing a Right-Opening Parabola
Graph \(x(t) = t^2 + 1\), \(y(t) = 2 + t\) for \(-3 \le t \le 3\). Indicate direction. OpenStax
T-Table
\(t\)\(-3\)\(-2\)\(-1\)\(0\)\(1\)\(2\)\(3\)
\(x\)105212510
\(y\)\(-1\)012345
Analysis
Parabola opening right; vertex \((1, 2)\) at \(t = 0\). As \(t\) increases the curve moves downward then rightward/upward.
Example 2
Graphing an Ellipse
Graph \(x = 2\cos t\), \(y = 4\sin t\) for \(0 \le t \le 2\pi\). Identify the curve and indicate direction. OpenStax
Key Points
\(t\)\(0\)\(\pi/2\)\(\pi\)\(3\pi/2\)\(2\pi\)
\(x\)20\(-2\)02
\(y\)040\(-4\)0
Result
Ellipse \(\dfrac{x^2}{4} + \dfrac{y^2}{16} = 1\); traced counterclockwise as \(t\) goes from \(0\) to \(2\pi\).
Reference
Projectile Motion Equations
For an object launched at speed \(v_0\), angle \(\theta\) above horizontal, from initial height \(h\):
Horizontal
\[x(t) = (v_0 \cos\theta)\,t\]
Vertical
\[y(t) = -\tfrac{1}{2}g\,t^2 + (v_0 \sin\theta)\,t + h\]
  • \(g =\) ______\(32\) ft/s² (US) or \(9.8\) m/s² (metric).
  • The coefficient of \(t^2\) in the vertical equation is \(\frac{1}{2}g\), so it is ______\(16\) (not 32) in ft/s² problems.
Example 3
Projectile Motion — Baseball
A baseball is hit at 140 ft/s at \(45^\circ\) from height 3 ft. (a) Write the parametric equations. (b) Position at \(t = 2\). (c) When does it land, and how far? (d) Does it clear a 10-ft fence at \(x = 400\) ft? OpenStax
(a) Write equations
\[v_0\cos 45^\circ = v_0\sin 45^\circ = 70\sqrt{2} \approx 98.99\]
\[x(t) = 70\sqrt{2}\,t, \qquad y(t) = -16t^2 + 70\sqrt{2}\,t + 3\]
(b) Position at \(t = 2\)
\[x(2) \approx 198 \;\text{ft}, \qquad y(2) = -64 + 197.98 + 3 \approx 137 \;\text{ft}\]
(c) Landing
\[y = 0: \quad -16t^2 + 98.99t + 3 = 0 \;\Longrightarrow\; t \approx 6.22 \;\text{sec}\]
Distance: \(x(6.22) \approx 615.7\) ft
(d) Fence clearance at \(x = 400\)
\[t = \frac{400}{70\sqrt{2}} \approx 4.04 \;\text{sec} \;\Longrightarrow\; y(4.04) \approx 141.8 \;\text{ft} > 10 \;\text{ft} \;\checkmark\]
Yes — the ball clears the fence by over 130 ft.
