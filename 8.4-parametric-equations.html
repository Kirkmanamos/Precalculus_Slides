8.4 Parametric Equations
Objective: Write and interpret parametric equations, eliminate the parameter, and model linear motion.
parameter orientation eliminating \(t\) trig identity method
Reference
Parametric Equations
Definition
\[x = f(t), \qquad y = g(t)\]
  • Each value of \(t\) gives an ordered pair \((x(t), y(t))\) on the curve.
  • As \(t\) increases, the curve is traced in a specific __________direction (called the orientation).
  • The parameter \(t\) often represents ________time, but it can be any indexing quantity.
  • Two parametric curves can trace the same \((x,y)\) set in ____________opposite directions — they are different curves.

Direction
Upward parabola \(y = x^2 - 1\); traced left-to-right as \(t\) increases.
Example 2
Linear Motion
An object travels at constant rate from \((-5, 3)\) to \((3, -1)\) in 4 seconds. Write parametric equations for its position. OpenStax
Find rates
\[x\text{-rate} = \frac{3 - (-5)}{4} = 2 \;\text{per sec} \qquad y\text{-rate} = \frac{-1 - 3}{4} = -1 \;\text{per sec}\]
Write equations & verify
\[\boxed{x(t) = 2t - 5, \quad y(t) = -t + 3}\]
Check: \(t = 0 \to (-5, 3)\;\checkmark\); \(t = 4 \to (3, -1)\;\checkmark\)
Reference
Eliminating the Parameter
Goal: Convert the parametric pair to a single equation in \(x\) and \(y\) only.
Algebraic Method
1. Solve the easier equation for ______\(t\).
2. Substitute into the other equation.
3. Simplify and check the domain.
Trig Identity Method
When \(x = a\cos t\) and \(y = b\sin t\):
Solve for \(\cos t\) and \(\sin t\), then use \(\cos^2 t + \sin^2 t =\) ____\(1\).
Domain Warning
The Cartesian equation may have a wider domain than the parametric curve — always verify.
Example 3
Eliminating the Parameter (Algebraic)
Eliminate the parameter: \(x(t) = t^2 + 1\), \(y(t) = 2 + t\). Identify the curve. OpenStax
Solve for \(t\)
\[y = 2 + t \;\Longrightarrow\; t = y - 2\]
Substitute
\[x = (y - 2)^2 + 1 \;\Longrightarrow\; \boxed{x = y^2 - 4y + 5}\]
Parabola opening right with vertex \((1, 2)\).
Examples 4 & 5
More Elimination Practice
Eliminate the parameter: (a) \(x = e^{-t},\; y = 3e^t\) and (b) \(x = \sqrt{t} + 2,\; y = \log t\). OpenStax
Part (a)
\[x = e^{-t} \;\Longrightarrow\; e^t = \frac{1}{x} \;\Longrightarrow\; y = \frac{3}{x}\]
\(\boxed{y = 3/x}\) — right branch only (\(x > 0\)) since \(e^{-t} > 0\).
Part (b)
\[x = \sqrt{t} + 2 \;\Longrightarrow\; t = (x - 2)^2 \;\Longrightarrow\; y = \log\!\big((x-2)^2\big)\]
\(\boxed{y = 2\log(x - 2)}\); domain \(x > 2\).
Example 6
Trig Identity Elimination
Eliminate the parameter from \(x(t) = 4\cos t,\; y(t) = 3\sin t,\; 0 \le t \le 2\pi\). OpenStax
Isolate trig functions
\[\cos t = \frac{x}{4}, \qquad \sin t = \frac{y}{3}\]
Apply Pythagorean Identity
\[\cos^2 t + \sin^2 t = 1 \;\Longrightarrow\; \left(\frac{x}{4}\right)^2 + \left(\frac{y}{3}\right)^2 = 1\]
\[\boxed{\frac{x^2}{16} + \frac{y^2}{9} = 1}\]
Ellipse with semi-major axis 4 (horizontal) and semi-minor axis 3 (vertical). Traced counterclockwise.
Concept

What Parametric Form Tells You

Parametric equations carry information that \(y = f(x)\) cannot:
Direction
The orientation — which way the curve is traced as \(t\) increases.
Position at Time \(t\)
The exact location of an object at a specific moment.
Self-Intersection Timing
Whether a curve crosses itself at different times.
