4.6 Graphs of Sine and Cosine
Objective: Sketch the parent graphs of \(y = \sin x\) and \(y = \cos x\), identify amplitude, period, phase shift, and vertical translation, and write trigonometric equations from a graph.

Concept
Anatomy of \(y = \sin x\)
The sine curve oscillates between \(-1\) and \(1\), crossing the \(x\)-axis at every multiple of \(\pi\), reaching a maximum at \(x = \pi/2 + 2\pi k\), and a minimum at \(x = -\pi/2 + 2\pi k\) (equivalently \(3\pi/2 + 2\pi k\)).
Where Sine Is Zero
At every integer multiple of \(\pi\): \(\ldots,\,-2\pi,\,-\pi,\,0,\,\pi,\,2\pi,\ldots\)
Where Sine Is ±1
Maxima at \(\pi/2 + 2\pi k\), minima at \(-\pi/2 + 2\pi k\). One full max-min pair per period.
Concept
Periodic Function — Period \(= 2\pi\)
A periodic function returns to the same value at regular intervals. The period is the smallest interval over which the function completes one full cycle. For \(\sin\) and \(\cos\), the period is \(2\pi\).
Concept
Amplitude — Half the Max-Minus-Min Distance
The amplitude measures how far the curve reaches above (or below) its midline. For \(y = \sin x\), the max is \(1\) and the min is \(-1\), so the amplitude is \(\tfrac{1 - (-1)}{2} = 1\).
Concept
Build the Graph of \(y = \cos x\)
Same procedure — sample standard angles, plot, extend. Cosine starts at \(1\) (not \(0\)) and is essentially the sine curve shifted left by \(\pi/2\).

Concept
Anatomy of \(y = \cos x\) — Same Amp, Same Period
Cosine has the same amplitude \(1\) and period \(2\pi\) as sine. The difference: cosine starts at its maximum, with maxes at \(0, \pm 2\pi, \ldots\), mins at \(\pm \pi\), and zeros at \(\pi/2 + \pi k\).

Reference
General Form — Four Parameters \(a, b, c, d\)
Every sinusoidal graph in this course can be written in one of these two forms. Each parameter controls one transformation of the parent curve.
Sine
\[y = a \sin\!\bigl(b(x - c)\bigr) + d\]
Cosine
\[y = a \cos\!\bigl(b(x - c)\bigr) + d\]
\(|a|\) — Amplitude
Vertical stretch. A negative \(a\) also reflects the curve across the \(x\)-axis.
\(b\) — Frequency
Period \(= \dfrac{2\pi}{|b|}\). Larger \(b\) compresses the wave horizontally; smaller \(b\) stretches it.
\(c\) — Phase Shift
Horizontal translation. Positive \(c\) shifts right; negative \(c\) shifts left.
\(d\) — Vertical Translation
Shifts the midline up by \(d\). The whole curve rides on \(y = d\) instead of \(y = 0\).
Reference
Two Parameterizations — Factor First!
Some textbooks write the inner expression as \((bx - c)\) instead of \(b(x - c)\). The phase shift differs between the two forms — always factor out \(b\) before reading \(c\).
This Class
\[y = a\sin\!\bigl(b(x - c)\bigr) + d \qquad \text{phase shift} = c\]
Some Books
\[y = a\sin(bx - c) + d \qquad \text{phase shift} = \dfrac{c}{b}\]
These Are Not the Same
\(y = \sin(2x - \pi) = \sin\!\bigl(2(x - \tfrac{\pi}{2})\bigr)\) ⇒ phase shift \(= \tfrac{\pi}{2}\)
\(y = \sin\!\bigl(2(x - \pi)\bigr)\) ⇒ phase shift \(= \pi\)
Always Factor
When you see \(bx - c\) inside a sine or cosine, factor out \(b\) first. Then read the phase shift directly off the \((x - c)\) form. A leading \(+\) inside means a shift to the left.
Example 1
Analyze \(y = 3\sin\!\left(\dfrac{x}{2}\right)\)
Find the amplitude, period, phase shift, and vertical translation of \(y = 3\sin\!\left(\dfrac{x}{2}\right)\).
Step 1 — Identify the Parameters
\[a = 3,\quad b = \dfrac{1}{2},\quad c = 0,\quad d = 0\]
Step 2 — Amplitude and Period
\[\text{amplitude} = |a| = 3 \qquad \text{period} = \dfrac{2\pi}{|b|} = \dfrac{2\pi}{1/2} = 4\pi\]
Step 3 — Phase Shift and Vertical Translation
\[\text{phase shift: none} \qquad \text{vertical translation: none}\]
Summary
amplitude \(= 3\), period \(= 4\pi\), no phase shift, no vertical translation.
Example 2
Analyze \(y = \dfrac{1}{2}\sin\!\left(x - \dfrac{\pi}{3}\right)\)
Find the amplitude, period, phase shift, and vertical translation of \(y = \dfrac{1}{2}\sin\!\left(x - \dfrac{\pi}{3}\right)\).
Step 1 — Identify the Parameters
\[a = \dfrac{1}{2},\quad b = 1,\quad c = \dfrac{\pi}{3},\quad d = 0\]
Step 2 — Amplitude and Period
\[\text{amplitude} = \dfrac{1}{2} \qquad \text{period} = \dfrac{2\pi}{1} = 2\pi\]
Step 3 — Phase Shift and Vertical Translation
\[\text{phase shift: } \dfrac{\pi}{3} \text{ to the right} \qquad \text{vertical translation: none}\]
\(c = +\pi/3\), so we read it as a shift right by \(\pi/3\).
Summary
amplitude \(= \tfrac{1}{2}\), period \(= 2\pi\), phase shift \(\tfrac{\pi}{3}\) right, no vertical translation.
Example 3
Analyze \(y = -3\cos(2\pi x + 4\pi)\)
Find the amplitude, period, phase shift, and vertical translation of \(y = -3\cos(2\pi x + 4\pi)\).
Step 1 — Factor Out \(b\) First
\[y = -3\cos\!\bigl(2\pi(x + 2)\bigr)\]
Pull \(2\pi\) out of the inner expression: \(2\pi x + 4\pi = 2\pi(x + 2)\).
Step 2 — Identify the Parameters
\[a = -3,\quad b = 2\pi,\quad c = -2,\quad d = 0\]
Inside the parentheses is \((x + 2) = (x - (-2))\), so \(c = -2\).
Step 3 — Amplitude (and Reflection)
\[\text{amplitude} = |a| = 3,\qquad \text{reflection in the } x\text{-axis (since } a < 0)\]
Step 4 — Period
\[\text{period} = \dfrac{2\pi}{|b|} = \dfrac{2\pi}{2\pi} = 1\]
Summary
amplitude \(= 3\) (reflected), period \(= 1\), phase shift \(= -2\) (\(2\) to the left), no vertical translation.
Example 4
Find the Equation from a Graph
The graph shown is one period of a sinusoidal curve of the form \(y = a\sin\!\bigl(b(x - c)\bigr)\). Find \(a\), \(b\), and \(c\).
The Graph
Step 1 — Amplitude and Reflection
\[\text{max} = 2, \;\text{min} = -2 \;\Longrightarrow\; \text{amplitude} = 2\]
The curve dips before it rises, so it's reflected → \(a = -2\). (Equivalently \(a = 2\) with the opposite phase shift — both work.)
Step 2 — Period and \(b\)
\[\text{period} = 2\pi \;\Longrightarrow\; \dfrac{2\pi}{b} = 2\pi \;\Longrightarrow\; b = 1\]
Step 3 — Phase Shift and Final Equation
A regular \(\sin\) curve crosses zero going up at \(x = 0\); this graph crosses zero going down at \(x = -\pi/2\). Shift left by \(\pi/2\): \(c = -\pi/2\).
\[y = -2\sin\!\left(x + \dfrac{\pi}{2}\right) \quad\text{or}\quad y = 2\sin\!\left(x - \dfrac{\pi}{2}\right)\]
Wrap-up

Sketch \(y = \sin x\) and \(y = \cos x\) over \([-2\pi, 2\pi]\) with correct intercepts, maxes, and mins.

From a sine or cosine graph, read off the amplitude, period, and key intercept locations.

In the form \(y = a\sin(b(x - c)) + d\), name what each of \(a\), \(b\), \(c\), \(d\) controls.

When you see \(bx + c\) inside a sine or cosine, factor out \(b\) before reading the phase shift.
Skill 5 — Reflection
Recognize that a negative \(a\) reflects the curve across the \(x\)-axis without changing the amplitude or period.
Skill 6 — Equation from Graph
Given a sinusoidal graph, reverse-engineer the equation by reading amplitude, period, and a key point.
