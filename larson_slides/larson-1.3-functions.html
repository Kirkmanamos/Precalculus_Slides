Unit 1 · Function Analysis
1.3 Functions
Objective: Decide whether a relation is a function, apply the vertical line test, and find the domain and range of various functions.
relation vs function vertical line test domain & range evaluating piecewise
Concept
Relation vs. Function
Relation
A relation is ________________a correspondence between two variables.
Function
A function is a relation where ________________no two ordered pairs have the same \(x\)-coordinate.
One input → one output
Every \(x\) has exactly one \(y\). Two points with the same \(x\) but different \(y\) values kill the "function" label.
Example 1
Which equations represent functions?
Solve each for \(y\). If you get exactly one value of \(y\) for each \(x\), it's a function.
a)
\[5x - 3y^2 = 9\]
b)
\[3y^3 + x = 1\]
c)
\[6|x| - 4x = y\]
d)
\[y = 5x^2 - 6x\]
e)
\[4x^2 + 4x - 3y^2 = 0\]
f)
\[x^2 = |y| + 6x\]
Example 1 · Verdicts
Solve for \(y\) and decide
a) \(5x - 3y^2 = 9\)
\(y = \pm\sqrt{\tfrac{5x-9}{3}}\)
✗ Not a function — two values of \(y\)
b) \(3y^3 + x = 1\)
\(y = \sqrt[3]{\tfrac{1-x}{3}}\)
✓ Function — cube root is single-valued
c) \(6|x| - 4x = y\)
\(y = 6|x| - 4x\)
✓ Function — already solved for \(y\)
d) \(y = 5x^2 - 6x\)
\(y = 5x^2 - 6x\)
✓ Function — classic quadratic
e) \(4x^2 + 4x - 3y^2 = 0\)
\(y = \pm\sqrt{\tfrac{4x^2+4x}{3}}\)
✗ Not a function — \(\pm\) gives two values
f) \(x^2 = |y| + 6x\)
\(|y| = x^2 - 6x\)
✗ Not a function — \(|y|\) allows \(\pm\)
Concept
Vertical Line Test
If a vertical line crosses a curve in more than one place at the same time, then ________________________the graph is not a function.
Passes (function)
Each vertical line crosses the parabola at most once.
Fails (not a function)
A vertical line crosses the sideways parabola twice.
Quick mental check
Slide an imaginary vertical line left to right. If it ever hits the graph more than once, it's not a function.

Concept
Input, Output, Domain, Range
Input · \(x\)
\(x\) is the input or ___________independent variable.
The \(x\)-values make up the _______domain.
Output · \(y\)
\(y\) is the output or ___________dependent variable.
The \(y\)-values make up the ______range.
  • Domain: every \(x\) you're allowed to plug in.
  • Range: every \(y\) the function can produce.
  • Two usual restrictions: can't divide by zero, can't take an even root of a negative.

Example 5
Domain and range of \(f(x) = \sqrt{9 - x^2}\)
The expression under the square root must be \(\ge 0\).
Step 1 · Radicand ≥ 0
\[9 - x^2 \ge 0 \ \Rightarrow\ x^2 \le 9 \ \Rightarrow\ -3 \le x \le 3\]
Domain
\[[-3,\ 3]\]
Step 2 · Find extreme outputs
\[\text{max at } x=0:\ \sqrt{9} = 3 \qquad \text{min at } x=\pm 3:\ 0\]
This is the top half of a circle of radius 3.
Range
\[[0,\ 3]\]
Example 6
Find the domain of \(f(x) = \dfrac{\sqrt{x-2}}{x-4}\)
Two restrictions: radicand \(\ge 0\) and denominator \(\ne 0\).
Step 1 · Radicand ≥ 0
\[x - 2 \ge 0 \ \Rightarrow\ x \ge 2\]
Step 2 · Denominator ≠ 0
\[x - 4 \ne 0 \ \Rightarrow\ x \ne 4\]
Step 3 · Combine
\[x \ge 2 \ \text{and}\ x \ne 4\]
Domain
\[[2,\ 4) \cup (4,\ \infty)\]
Example 7
Will the baseball clear the fence?
The height of a baseball hit is \(f(x) = -0.0032x^2 + x + 3\), where \(x\) is the horizontal distance from home plate (ft). Will it clear a 10-ft fence at \(x = 300\) ft?
Step 1 · Evaluate \(f(300)\)
\[f(300) = -0.0032(300)^2 + 300 + 3\]
Step 2 · Compute
\[= -0.0032(90{,}000) + 303 = -288 + 303\]
Step 3 · Height at 300 ft
\[f(300) = 15 \text{ feet}\]
Conclusion
\[15 > 10 \ \Rightarrow\ \text{Yes, the ball clears the fence.}\]
Clears by 5 feet.

Example 9
Evaluate the piecewise function
\[f(x) = \begin{cases} -(x+2)^2 + 4, & x \le -2 \\ \tfrac{1}{4}x + \tfrac{9}{2}, & x > -2 \end{cases}\]
a) \(f(-3)\) · which piece?
\[-3 \le -2 \ \Rightarrow\ \text{use top piece}\]
a) \(f(-3)\)
\[-(-3+2)^2 + 4 = -(1) + 4 = 3\]
b) \(f(8)\) · which piece?
\[8 > -2 \ \Rightarrow\ \text{use bottom piece}\]
b) \(f(8)\)
\[\tfrac{1}{4}(8) + \tfrac{9}{2} = 2 + \tfrac{9}{2} = \tfrac{13}{2}\]
c) \(f(-2)\) · boundary!
\[-2 \le -2 \ \Rightarrow\ \text{top piece (includes equality)}\]
Watch the \(\le\) vs \(>\) — the boundary goes to whichever piece has the equals sign.
c) \(f(-2)\)
\[-(-2+2)^2 + 4 = -(0) + 4 = 4\]
Summary
Big Ideas · 1.3
Function test (algebra)
Solve for \(y\). If you get \(\pm\) or \(|y|\), it's not a function.
Function test (graph)
The vertical line test: any vertical line hits the graph at most once.
Domain restrictions
No zero denominators. No negative radicands under even roots.
Range from graphs
Look at the lowest and highest \(y\)-values the curve reaches.
Evaluating piecewise
First pick the piece that contains your \(x\). Boundary goes to the piece with \(\le\) or \(\ge\).
Real-world models
Plug in the question's \(x\), compute \(f(x)\), and compare to a threshold.
