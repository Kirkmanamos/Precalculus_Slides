Unit 1 · Function Analysis
1.7 Inverse Functions
Objective: Find and verify inverse functions, use graphs and the horizontal line test to decide when inverses are functions, and identify one-to-one functions.
inverse reflection over \(y=x\) one-to-one horizontal line test
Concept
The inverse “undoes” the function
Notation
The inverse of \(f(x)\) is written \(f^{-1}(x)\) (“\(f\) inverse”). Note: this is not \(1/f(x)\).
Swap domain and range
The domain of \(f\) becomes the range of \(f^{-1}\), and vice versa. Inputs and outputs trade places.
Graph relationship
The graph of \(f^{-1}(x)\) is the reflection of \(f(x)\) over the line \(y = x\).
Method
Find an inverse algebraically
Step 1
Replace \(f(x)\) with \(y\).
Step 2
Swap \(x\) and \(y\).
Step 3
Solve the new equation for \(y\).
Step 4
Rewrite \(y\) as \(f^{-1}(x)\). Check: \(f(f^{-1}(x)) = x\) and \(f^{-1}(f(x)) = x\).
Example 1
Find the inverse of \(f(x) = x^3 - 4\)
Step 1 · Replace
\(y = x^3 - 4\)
Step 2 · Swap
\(x = y^3 - 4\)
Step 3 · Solve for \(y\)
\(x + 4 = y^3 \;\Rightarrow\; y = \sqrt[3]{x + 4}\)
Answer
\(f^{-1}(x) = \sqrt[3]{x + 4}\)
Check \(f(f^{-1}(x))\)
\(\bigl(\sqrt[3]{x+4}\bigr)^3 - 4 = (x + 4) - 4 = x\) ✓
Example 2
Find the inverse of \(f(x) = \dfrac{3x + 1}{x - 2}\)
Swap \(x\) and \(y\)
\(x = \dfrac{3y + 1}{y - 2}\)
Clear the denominator
\(x(y - 2) = 3y + 1\)
Distribute
\(xy - 2x = 3y + 1\)
Collect \(y\) on one side
\(xy - 3y = 2x + 1\)
Factor and solve
\(y(x - 3) = 2x + 1 \;\Rightarrow\; y = \dfrac{2x + 1}{x - 3}\)
Answer
\(f^{-1}(x) = \dfrac{2x + 1}{x - 3}\)
Example 3
Verify \(f\) and \(g\) are inverses
Given
\(f(x) = \dfrac{4}{x - 3}\) and \(g(x) = \dfrac{4}{x} + 3\). Show \(f(g(x)) = x\) and \(g(f(x)) = x\).
Compute \(f(g(x))\)
\(f\!\left(\dfrac{4}{x} + 3\right) = \dfrac{4}{\left(\tfrac{4}{x} + 3\right) - 3}\)
Simplify
\(= \dfrac{4}{\tfrac{4}{x}} = 4 \cdot \dfrac{x}{4} = x\) ✓
Compute \(g(f(x))\)
\(g\!\left(\dfrac{4}{x - 3}\right) = \dfrac{4}{\tfrac{4}{x - 3}} + 3\)
Simplify
\(= (x - 3) + 3 = x\) ✓
Conclusion
Both compositions equal \(x\), so \(f\) and \(g\) are inverses of each other.
Concept
The graph of \(f^{-1}\) is a reflection over \(y = x\)
Point swap
If \((a, b)\) is on \(f\), then \((b, a)\) is on \(f^{-1}\). Inputs and outputs trade places.
Fold the paper along \(y = x\) — the two graphs land on each other.
Concept
Horizontal line test & one-to-one
Vertical line test (review)
If a graph passes the vertical line test, it is a function (each \(x\) gives one \(y\)).
Horizontal line test
If a graph passes the horizontal line test, its inverse is a function — each \(y\) comes from only one \(x\).
One-to-one
A function is one-to-one when no two \(x\)-values in the domain give the same \(y\). The \(y\)’s don’t repeat.
A function has an inverse that is also a function iff it is one-to-one.
Example 4
Is each function one-to-one?
a) \(f(x) = 2x^2 - 4\)
NO — fails HLT
b) \(g(x) = \tfrac{1}{2}x^3 - 2\)
YES — passes HLT
c) \(h(x) = \sqrt{x} + 1\)
YES — passes HLT
Quick eye-test
Any graph that turns back on itself (like a parabola) fails the horizontal line test. Monotonic curves (always increasing or always decreasing) pass.

Example 6
Find the inverse of \(f(x) = \sqrt{2x - 3}\)
Swap
\(y = \sqrt{2x - 3}\) becomes \(x = \sqrt{2y - 3}\)
Square both sides
\(x^2 = 2y - 3\)
Solve for \(y\)
\(y = \dfrac{x^2 + 3}{2}\)
Restrict the domain
Since \(f(x) = \sqrt{2x-3} \geq 0\), the range of \(f\) is \([0, \infty)\). That becomes the domain of \(f^{-1}\): \(x \geq 0\).
Answer
\(f^{-1}(x) = \dfrac{x^2 + 3}{2}, \quad x \geq 0\)
Review
Summary of inverse functions
Notation
The inverse of \(f(x)\) is written \(f^{-1}(x)\).
Graph
The graph of \(f^{-1}(x)\) is the reflection of \(f(x)\) over the line \(y = x\).
Algebra
Switch \(x\) and \(y\), then solve for \(y\). Write with \(f^{-1}(x)\) notation.
