4.5 Trig Functions of Any Angle
Objective: Use reference angles to evaluate trigonometric functions of any angle, including angles whose terminal sides pass through non-unit-circle points.
reference angles terminal-side points \(r = \sqrt{x^2 + y^2}\) solving for \(\theta\) two-quadrant answers
Concept
From Q1 to Any Angle
You already know how to evaluate trig functions for angles in Q1, and for any multiple of 30°, 45°, or 60° via the unit circle. The trick for everything else is to relate it back to a Q1 angle — its reference angle.
A Point on the Terminal Side
What's New in 4.5
Trig works for any terminal-side point — the point doesn't have to live on the unit circle.
Strategy
Find the reference angle (the Q1 partner), evaluate the trig function there, then attach the correct sign from ASTC.
Concept
The Reference Angle \(\theta'\)
The reference angle \(\theta'\) is the positive acute angle between the terminal side of \(\theta\) and the \(x\)-axis (whichever side is closer). It always lives in \([0,\pi/2]\).
Quadrant I
Quadrant II
Quadrant III
Quadrant IV
Why It Matters
Once you have \(\theta'\), every trig value of \(\theta\) equals \(\pm\) the same trig value of \(\theta'\). The sign comes from which quadrant \(\theta\) lives in (ASTC).
Reference
Reference-Angle Formulas
For an angle \(\theta\) in standard position (already reduced to \([0^\circ, 360^\circ)\) or \([0, 2\pi)\)), use the formula matching its quadrant.
Quadrant I
\[\theta' = \theta\]
Quadrant II
\[\theta' = 180° - \theta \qquad \text{(or } \pi - \theta\text{)}\]
Quadrant III
\[\theta' = \theta - 180° \qquad \text{(or } \theta - \pi\text{)}\]
Quadrant IV
\[\theta' = 360° - \theta \qquad \text{(or } 2\pi - \theta\text{)}\]
First, Normalize the Angle
If \(\theta\) is negative or larger than one full revolution, add or subtract \(360^\circ\) (or \(2\pi\)) until you land in \([0^\circ, 360^\circ)\) — then apply the formula.
Example 1
Reference Angle for \(330°\)
Find the reference angle for \(\theta = 330°\).
Step 1 — Identify the Quadrant
\[270° < 330° < 360° \quad\Longrightarrow\quad \theta \text{ is in Quadrant IV}\]
Step 2 — Apply the Q4 Formula
\[\theta' = 360° - 330° = 30°\]
Reference Angle
\[\theta' = 30°\]
Example 2
Reference Angle for \(-\dfrac{5\pi}{6}\)
Find the reference angle for \(\theta = -\dfrac{5\pi}{6}\).
Step 1 — Find a Coterminal Angle
\[-\dfrac{5\pi}{6} + 2\pi = -\dfrac{5\pi}{6} + \dfrac{12\pi}{6} = \dfrac{7\pi}{6}\]
Step 2 — Identify the Quadrant
\[\pi < \dfrac{7\pi}{6} < \dfrac{3\pi}{2} \quad\Longrightarrow\quad \text{Quadrant III}\]
Step 3 — Apply the Q3 Formula
\[\theta' = \dfrac{7\pi}{6} - \pi = \dfrac{7\pi}{6} - \dfrac{6\pi}{6} = \dfrac{\pi}{6}\]
Reference Angle
\[\theta' = \dfrac{\pi}{6}\]
Example 3
Reference Angle for \(\theta = 2.3\) Radians
Find the reference angle for \(\theta = 2.3\) (radians, not a multiple of \(\pi\)).
Step 1 — Locate the Radian Angle
\[\dfrac{\pi}{2} \approx 1.571 \;\;<\;\; 2.3 \;\;<\;\; \pi \approx 3.142\]
So \(\theta\) lies in Quadrant II.
Step 2 — Apply the Q2 Formula
\[\theta' = \pi - 2.3 \approx 3.1416 - 2.3 = 0.8416\]
Reference Angle
\[\theta' \approx 0.8416 \text{ rad}\]
Concept
Trig from Any Point on the Terminal Side
If the terminal side of \(\theta\) passes through \(P(x, y)\), the same six trig values come from the ratios of \(x\), \(y\), and \(r = \sqrt{x^2 + y^2}\). The point doesn't have to be on the unit circle.
Triangle with Legs \(x\), \(y\) and Hypotenuse \(r\)
Distance to Origin
\[r = \sqrt{x^{2} + y^{2}}\]
Pythagorean theorem on the legs \(x\) and \(y\). \(r\) is always positive.
Why It Generalizes the Unit Circle
On the unit circle, \(r = 1\), so the ratios collapse to \(\sin\theta = y\), \(\cos\theta = x\). For any other point, you just divide by \(r\).
Reference
Six Trig Functions in Terms of \(x\), \(y\), \(r\)
These match the SOHCAHTOA ratios you already know, but now the legs are signed: \(x\) and \(y\) carry the sign of the quadrant.
Primary
\[\sin\theta = \dfrac{y}{r} \qquad \cos\theta = \dfrac{x}{r} \qquad \tan\theta = \dfrac{y}{x}\]
Reciprocals
\[\csc\theta = \dfrac{r}{y} \qquad \sec\theta = \dfrac{r}{x} \qquad \cot\theta = \dfrac{x}{y}\]
Setup
\[x^{2} + y^{2} = r^{2} \quad\text{with}\quad r > 0\]
Watch the Signs
\(r\) is always positive. The signs of sin/cos/tan come entirely from the signs of \(x\) and \(y\) — which depend on the quadrant.
Same Six Values for the Whole Ray
Pick any point on the terminal side: the ratios \(y/r\), \(x/r\), \(y/x\) are the same. The trig value depends on the angle, not on which point you pick.
Example 4
Six Trig Functions Given a Terminal-Side Point
The terminal side of \(\theta\) in standard position contains the point \((8,\,-15)\). Find the exact values of all six trig functions of \(\theta\).
Step 1 — Identify \(x\), \(y\), and the Quadrant
\[x = 8,\; y = -15 \quad\Longrightarrow\quad \theta \text{ is in Quadrant IV}\]
Step 2 — Find \(r\) (Pythagorean Theorem)
\[r = \sqrt{8^{2} + (-15)^{2}} = \sqrt{64 + 225} = \sqrt{289} = 17\]
8-15-17 is a Pythagorean triple, like 3-4-5 or 5-12-13.
Step 3 — Build the Six Ratios
\[\sin\theta = -\dfrac{15}{17} \qquad \cos\theta = \dfrac{8}{17} \qquad \tan\theta = -\dfrac{15}{8}\]
\[\csc\theta = -\dfrac{17}{15} \qquad \sec\theta = \dfrac{17}{8} \qquad \cot\theta = -\dfrac{8}{15}\]
Q4 Sign Check
In Q4, cosine and secant are positive; the other four are negative — matches ASTC. ✓
Example 5
Given One Sign Plus One Value, Find the Rest
Given \(\sin\theta = -\dfrac{2}{3}\) and \(\tan\theta > 0\), find \(\cos\theta\) and \(\cot\theta\).
Step 1 — Determine the Quadrant
\(\sin\theta < 0 \Rightarrow y < 0\) (Q3 or Q4)
\(\tan\theta > 0 \Rightarrow x\) and \(y\) have the same sign (Q1 or Q3)
Intersection: Quadrant III
Step 2 — Read \(y\) and \(r\) from sin
\[\sin\theta = \dfrac{y}{r} = \dfrac{-2}{3} \quad\Longrightarrow\quad y = -2,\; r = 3\]
Step 3 — Solve for \(x\)
\[x^{2} + (-2)^{2} = 3^{2} \;\Longrightarrow\; x^{2} = 5 \;\Longrightarrow\; x = \pm\sqrt{5}\]
In Q3 the \(x\)-coordinate is negative, so \(x = -\sqrt{5}\).
Step 4 — Compute cos and cot
\[\cos\theta = \dfrac{x}{r} = -\dfrac{\sqrt{5}}{3} \qquad \cot\theta = \dfrac{x}{y} = \dfrac{-\sqrt{5}}{-2} = \dfrac{\sqrt{5}}{2}\]
Final Answers
\[\cos\theta = -\dfrac{\sqrt{5}}{3} \qquad \cot\theta = \dfrac{\sqrt{5}}{2}\]
Concept
Given a Trig Value, Find All Angles
When you're handed a trig value instead of an angle, expect two answers in one revolution — one in each of the two quadrants where the function has that sign.
Step 1
Find the reference angle \(\theta'\) by taking the inverse trig function of the absolute value.
Step 2
Identify the two quadrants where the given trig function has the given sign (via ASTC).
Step 3
Build each angle from \(\theta'\) using the same quadrant formulas as before:
Q2: \(180^\circ - \theta'\)  ·  Q3: \(180^\circ + \theta'\)  ·  Q4: \(360^\circ - \theta'\)
Why Absolute Value?
Calculator's inverse functions return only one angle. Stripping the sign forces \(\theta'\) into Q1, then you control where it lives using ASTC — instead of letting the calculator decide.
Example 6
Given \(\tan\theta = -0.4623\), Find \(\theta\) in \([0^\circ, 360^\circ)\)
If \(\tan\theta = -0.4623\) and \(0^\circ \le \theta < 360^\circ\), find \(\theta\) to the nearest \(0.1^\circ\).
Step 1 — Reference Angle (drop the sign)
\[\theta' = \tan^{-1}(0.4623) \approx 24.8°\]
Step 2 — Where Is Tangent Negative?
ASTC: tangent is positive in Q1 and Q3, so it's negative in Q2 and Q4.
Step 3 — Build Both Angles from the Reference Angle
\[\theta_{\text{Q2}} = 180° - 24.8° = 155.2° \qquad \theta_{\text{Q4}} = 360° - 24.8° = 335.2°\]
Final Answers
\[\theta \approx 155.2° \quad\text{or}\quad \theta \approx 335.2°\]
Example 7
Given \(\cos\theta = -0.3842\), Find \(\theta\) in \([0, 2\pi)\)
If \(\cos\theta = -0.3842\) and \(0 \le \theta < 2\pi\), find \(\theta\) to the nearest \(0.0001\) radian.
Step 1 — Reference Angle
\[\theta' = \cos^{-1}(0.3842) \approx 1.1765\]
Step 2 — Where Is Cosine Negative?
ASTC: cosine is positive in Q1 and Q4, so it's negative in Q2 and Q3.
Step 3 — Build Both Angles from the Reference Angle
\[\theta_{\text{Q2}} = \pi - 1.1765 \approx 1.9651 \qquad \theta_{\text{Q3}} = \pi + 1.1765 \approx 4.3181\]
Final Answers
\[\theta \approx 1.9651 \quad\text{or}\quad \theta \approx 4.3181 \text{ rad}\]
Wrap-up
By the End of 4.5 You Should Be Able To...
Check yourself against each of these. If anything feels shaky, revisit the relevant example before the Unit 4 review.
Skill 1 — Reference Angles
Find the reference angle for any \(\theta\) in degrees or radians by identifying its quadrant and applying the right formula.
Skill 2 — Coterminal First
Reduce a negative or oversized angle to the interval \([0^\circ, 360^\circ)\) or \([0, 2\pi)\) before applying the reference-angle formula.
Skill 3 — Trig from a Point
Given any point on the terminal side, use \(r = \sqrt{x^2 + y^2}\) to compute all six trig values.
Skill 4 — Two Constraints, One Quadrant
Use sign clues (like "sine negative AND tangent positive") to pin down which quadrant \(\theta\) is in, then find the missing trig values.
Skill 5 — Solve for Angles
Given a trig value, find the reference angle from its absolute value, then locate both angles in \([0^\circ, 360^\circ)\) using ASTC.
Skill 6 — Expect Two Answers
When solving sin/cos/tan = (value), there are two angles in one revolution. Don't stop at the first one.
