5.1 Using Fundamental Identities

Simplifying and factoring trigonometric expressions

$\sin^2 x + \cos^2 x = 1$ $1 + \tan^2 x = \sec^2 x$ $1 + \cot^2 x = \csc^2 x$
REFERENCE

Trigonometric Identities

Reciprocal Identities
$\sin\theta = \dfrac{1}{\csc\theta}$ $\csc\theta = \dfrac{1}{\sin\theta}$
$\cos\theta = \dfrac{1}{\sec\theta}$ $\sec\theta = \dfrac{1}{\cos\theta}$
$\tan\theta = \dfrac{1}{\cot\theta}$ $\cot\theta = \dfrac{1}{\tan\theta}$
Also work with powers…
$\sin^2\theta = \dfrac{1}{\csc^2\theta}$
Quotient Identities
$\tan\theta = \dfrac{\sin\theta}{\cos\theta}$ $\cot\theta = \dfrac{\cos\theta}{\sin\theta}$
Pythagorean Identities
$\cos^2\theta + \sin^2\theta = 1$
$\tan^2\theta + 1 = \sec^2\theta$
$\cot^2\theta + 1 = \csc^2\theta$
Cofunction Identities
$\sin(\theta) = \cos(90° - \theta)$
$\cos(\theta) = \sin(90° - \theta)$
$\tan(\theta) = \cot(90° - \theta)$
$\cot(\theta) = \tan(90° - \theta)$
$\sec(\theta) = \csc(90° - \theta)$
$\csc(\theta) = \sec(90° - \theta)$
↑ $\theta$ is an acute angle ↑
REFERENCE

Even / Odd Identities

Even Functions

  • $\cos(-x) = \phantom{-}\cos x$
  • $\sec(-x) = \phantom{-}\sec x$

Odd Functions

  • $\sin(-x) = -\sin x$
  • $\tan(-x) = -\tan x$
  • $\csc(-x) = -\csc x$
  • $\cot(-x) = -\cot x$

EXAMPLE 1 · SIMPLIFY

$\sin^2 x \sec x - \sec x$

Factor
$= \sec x\,(\sin^2 x - 1)$
Pythagorean Id.
$= \sec x\,(-\cos^2 x)$ since $\sin^2 x - 1 = -\cos^2 x$
Answer
$= \dfrac{-\cos^2 x}{\cos x} = -\cos x$

EXAMPLE 2 · SIMPLIFY

$\sin^2 x - \cos^2 x$

Substitute
Replace $\cos^2 x$ with $1 - \sin^2 x$:
$= \sin^2 x - (1 - \sin^2 x)$
Answer
$= 2\sin^2 x - 1$ (equivalently, $\,1 - 2\cos^2 x$)

EXAMPLE 3 · FACTOR

$6\cot^2\!\theta - 13\cot\theta + 5$

Substitution
Let $u = \cot\theta$. Now factor $6u^2 - 13u + 5$.
Find factors
Need two factors of $6 \cdot 5 = 30$ that sum to $-13$: $-10$ and $-3$
Rewrite middle
$= 6u^2 - 10u - 3u + 5$
Group
$= 2u(3u - 5) - 1(3u - 5)$
Factor
$= (2u - 1)(3u - 5)$
Answer
$= (2\cot\theta - 1)(3\cot\theta - 5)$

EXAMPLE 4 · FACTOR

$\cot^2 x - 2\csc x - 7$

Pythagorean Id.
Use $\cot^2 x = \csc^2 x - 1$:
$= \csc^2 x - 1 - 2\csc x - 7$
Combine
$= \csc^2 x - 2\csc x - 8$
Answer
$= (\csc x - 4)(\csc x + 2)$

Click to reveal next step

EXAMPLE 5 · SIMPLIFY

$\csc x \sec x - \sin x \sec x$

Factor
$= \sec x\,(\csc x - \sin x)$
Reciprocal Ids.
$= \dfrac{1}{\cos x}\!\left(\dfrac{1}{\sin x} - \sin x\right)$
$= \dfrac{1}{\cos x} \cdot \dfrac{1 - \sin^2 x}{\sin x}$
Pythagorean Id.
$= \dfrac{1}{\cos x} \cdot \dfrac{\cos^2 x}{\sin x}$ since $1 - \sin^2 x = \cos^2 x$
Answer
$= \dfrac{\cos x}{\sin x} = \cot x$

Click to reveal next step

EXAMPLE 6 · REWRITE

Rewrite $\dfrac{\sin^2 x}{1 + \cos x}$ without a fraction.

Pythagorean Id.
$\sin^2 x = 1 - \cos^2 x = (1 - \cos x)(1 + \cos x)$
Substitute
$= \dfrac{(1 - \cos x)(1 + \cos x)}{1 + \cos x}$
Answer
$= 1 - \cos x$

EXAMPLE 7 · SUBSTITUTION

Use $x = 9\cos\theta$, $\,0 < \theta < \dfrac{\pi}{2}$, to write $\sqrt{81 - x^2}$ as a trig function of $\theta$.

Substitute
$x = 9\cos\theta \;\Longrightarrow\; x^2 = 81\cos^2\!\theta$
Simplify radicand
$81 - x^2 = 81 - 81\cos^2\!\theta = 81(1 - \cos^2\!\theta)$
Pythagorean Id.
$= 81\sin^2\!\theta$ since $1 - \cos^2\theta = \sin^2\theta$
Answer
$\sqrt{81 - x^2} = \sqrt{81\sin^2\!\theta} = 9\sin\theta$ positive since $0 < \theta < \tfrac{\pi}{2}$

SUMMARY

Strategies for Simplifying

1
Factor out a common term
Look for a shared factor across all terms and pull it out front. Ex 1, 5
2
Apply a Pythagorean Identity
Swap $\sin^2 x + \cos^2 x = 1$ (or its rearrangements) to reduce terms. Ex 1, 2, 4, 5, 6, 7
3
Rewrite in sine & cosine
Convert $\sec$, $\csc$, $\tan$, $\cot$ to $\dfrac{1}{\cos}$, $\dfrac{1}{\sin}$, etc. to find cancellations. Ex 1, 5
4
$u$-substitution for factoring
Replace a trig expression with $u$ to reveal a factorable polynomial. Ex 3
5
Factor as difference of squares
Write $1 - \cos^2 x$ as $(1 - \cos x)(1 + \cos x)$ to cancel with the denominator. Ex 6
6
Cancel common factors
After rewriting, look for matching factors in numerator and denominator to simplify. Ex 1, 5, 6

Most problems use more than one strategy — look for the first move that reduces complexity.
