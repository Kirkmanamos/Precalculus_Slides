Chapter 1 · Functions
1.1 Functions and Function Notation
Objective: Determine whether a relation represents a function, use function notation to evaluate functions, and represent functions using tables, mapping diagrams, and graphs.
relations & functions function notation vertical line test evaluating one-to-one
Definition
What Is a Function?
OpenStax
Relation
A relation is ________________a set of ordered pairs. The set of first coordinates is the domain; the set of second coordinates is the range.
Function
A function is a relation that assigns to each element in the domain ________________exactly one element in the range.
Key idea — One input → one output
Every \(x\) in the domain has exactly one corresponding \(y\). Two different inputs can share the same output, but one input cannot produce two different outputs.
Example 1
Is this relation a function?
OpenStax
Determine whether each relation represents a function.
a) Set of ordered pairs
\[\{(1,\,5),\ (2,\,6),\ (3,\,7),\ (4,\,8)\}\]
✓ Function — each input maps to exactly one output
b) Set of ordered pairs
\[\{(1,\,5),\ (1,\,6),\ (3,\,7),\ (4,\,8)\}\]
✗ Not a function — the input \(1\) maps to both \(5\) and \(6\)
Check
Scan the first coordinates. If any \(x\)-value repeats with a different \(y\)-value, it's not a function.
Concept
Mapping Diagrams
OpenStax
Function ✓
Each input has one arrow leaving it.
Not a Function ✗
Input \(1\) has two arrows — maps to both A and B.
Quick Check
Spot the non-function
Which relation is not a function?
Correct. In B the input \(1\) maps to both \(5\) and \(6\) — one input can’t have two outputs.
Look for a repeated input with different outputs. In B, \(1\) maps to both \(5\) and \(6\). (C is fine — different inputs sharing one output is allowed.)
Concept
Function Notation
OpenStax
The notation \(f(x)\)
Instead of writing \(y\), we write ________\(f(x)\).
Read as "________________\(f\) of \(x\)" or "the value of \(f\) at \(x\)."
The letter \(f\) names the function. The letter \(x\) is the input.
Why this matters
\(y = x^2 + 1\) and \(f(x) = x^2 + 1\) say the same thing, but function notation tells us:
• Which function (by name: \(f\), \(g\), \(h\), …)
• What input to use: \(f(\textcolor{#dc2626}{3})\) means "plug in \(\textcolor{#dc2626}{3}\)"
Example 2
Evaluate \(f(x) = x^2 - 3x + 5\)
OpenStax
Find each value.
a) \(f(2)\)
\[f(2) = (2)^2 - 3(2) + 5 = 4 - 6 + 5 = 3\]
b) \(f(-1)\)
\[f(-1) = (-1)^2 - 3(-1) + 5 = 1 + 3 + 5 = 9\]
c) \(f(a)\)
\[f(a) = a^2 - 3a + 5\]
Replace every \(x\) with \(a\). The expression stays unevaluated.
d) \(f(a + h)\)
\[f(a+h) = (a+h)^2 - 3(a+h) + 5\]
d) expanded
\[= a^2 + 2ah + h^2 - 3a - 3h + 5\]
This form is essential for the difference quotient later in the course.
Quick Check
Mind the signs
For \(f(x) = x^2 - 3x + 5\), what is \(f(-2)\)?
Correct. \(f(-2) = (-2)^2 - 3(-2) + 5 = 4 + 6 + 5 = 15\).
Two sign traps: \((-2)^2 = 4\) (not \(-4\)) and \(-3(-2) = +6\). So \(f(-2) = 4 + 6 + 5 = 15\).
Concept
Vertical Line Test
A set of points in the plane is the graph of a function if and only if every vertical line intersects the graph in ________________________at most one point.
Passes (function)
Each vertical line crosses the parabola at most once.
Fails (not a function)
A vertical line crosses the circle twice.
Quick mental check
Slide an imaginary vertical line left to right. If it ever hits the graph more than once, it's not a function.

Example 5
Will the baseball clear the fence?
Larson
The height of a baseball hit is \(f(x) = -0.0032x^2 + x + 3\), where \(x\) is the horizontal distance from home plate (ft). Will it clear a 10-ft fence at \(x = 300\) ft?
Step 1 · Evaluate \(f(300)\)
\[f(300) = -0.0032(300)^2 + 300 + 3\]
Step 2 · Compute
\[= -0.0032(90{,}000) + 303 = -288 + 303\]
Step 3 · Height at 300 ft
\[f(300) = 15 \text{ feet}\]
Conclusion
\[15 > 10 \ \Rightarrow\ \text{Yes, the ball clears the fence by 5 feet.}\]
Concept
One-to-One Functions
OpenStax
Definition
A function is one-to-one if each element of the range corresponds to ________________exactly one element of the domain.
Equivalently: if \(f(a) = f(b)\), then ________\(a = b\).
One-to-one ✓
\(f(x) = x^3\) — different inputs always produce different outputs. No \(y\)-value repeats.
Not one-to-one ✗
\(f(x) = x^2\) — both \(f(2) = 4\) and \(f(-2) = 4\). Two inputs share an output.
Quick Check
Function vs. one-to-one
Is \(f(x) = x^2\) a one-to-one function?
Correct. \(x^2\) is a function, but one-to-one is stricter: \(f(2)=f(-2)=4\) means two inputs share an output.
It is a function (passes the VLT), but one-to-one is stronger: \(f(2)=f(-2)=4\), so two different inputs share an output — not one-to-one.
Example 6
Which equations represent functions?
Larson
Solve each for \(y\). If you get exactly one value of \(y\) for each \(x\), it's a function.
a) \(5x - 3y^2 = 9\)
\(y = \pm\sqrt{\tfrac{5x-9}{3}}\)
✗ Not a function — two values of \(y\)
b) \(3y^3 + x = 1\)
\(y = \sqrt[3]{\tfrac{1-x}{3}}\)
✓ Function — cube root is single-valued
c) \(y = 5x^2 - 6x\)
Already solved for \(y\)
✓ Function — classic quadratic
d) \(x^2 = |y| + 6x\)
\(|y| = x^2 - 6x\)
✗ Not a function — \(|y|\) allows \(\pm\)
Summary
Big Ideas · 1.1
Function definition
A function assigns to each input (domain) exactly one output (range). One input → one output.
Function notation
\(f(x)\) names the function and specifies the input. Evaluate by substituting the input value for \(x\).
Vertical Line Test
If any vertical line crosses the graph more than once, it's not a function.
Representations
Functions can be given as equations, tables, mapping diagrams, graphs, or verbal descriptions.
One-to-one
Each output comes from exactly one input. Important for inverse functions (Section 1.7).
Evaluating
Replace every \(x\) with the given input — numbers, variables, or expressions like \(a + h\).
