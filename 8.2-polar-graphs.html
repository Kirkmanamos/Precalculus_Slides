Reference
Symmetry Tests for Polar Graphs
Symmetry helps reduce the number of points needed in a t-table. These tests are sufficient but not necessary.
Symmetry type Substitution Condition for symmetry
About the Polar Axis (x-axis) Replace \((r, \theta)\) with \((r, -\theta)\) ________________________Equation is unchanged
About the Line \(\theta = \frac{\pi}{2}\) (y-axis) Replace \((r, \theta)\) with \((-r, -\theta)\) ________________________Equation is unchanged
About the Pole (origin) Replace \((r, \theta)\) with \((-r, \theta)\) ________________________Equation is unchanged
Caution
A curve can be symmetric even if its symmetry test is inconclusive!
Example 1
Testing Symmetry
Test the polar equation \(r = 2\sin\theta\) for symmetry about the line \(\theta = \frac{\pi}{2}\) (the y-axis). OpenStax
Substitute \((-r, -\theta)\)
\[-r = 2\sin(-\theta)\]
\[-r = -2\sin\theta \;\Longrightarrow\; \mathbf{r = 2\sin\theta}\]
Since \(\sin(-\theta) = -\sin\theta\) (odd function), the negatives cancel out, leaving the original equation unchanged.
Conclusion: Symmetric about \(\theta = \frac{\pi}{2}\).
Concept

Graphing a Polar Equation

Follow these structured steps to sketch any polar curve:
1. Test Symmetry
Apply the x-axis, y-axis, and pole symmetry tests.
2. Find Zeros & Max
Set \(r = 0\) to find pole intercepts. Identify maximum \(|r|\) value.
3. Build T-Table
Compute values at key angles (\(\frac{\pi}{6}, \frac{\pi}{4}, \frac{\pi}{3}, \frac{\pi}{2}\)).
4. Plot & Reflect
Plot points and reflect across the line(s) of symmetry.
Example 2
Graphing \(r = 2\sin\theta\)
Analyze and sketch the graph of \(r = 2\sin\theta\). Find zeros and the maximum modulus. OpenStax
Find Key Features
Symmetry: About y-axis (tested in Ex 1).
Zeros: \(2\sin\theta = 0 \;\Longrightarrow\; \theta = 0, \pi\).
Maximum: Max \(|r| = 2\) occurs when \(\theta = \frac{\pi}{2}\).
T-Table & Curve Type
T-Table: \(\theta = 0 \to r=0\); \(\frac{\pi}{6} \to 1\); \(\frac{\pi}{4} \to \sqrt{2} \approx 1.4\); \(\frac{\pi}{3} \to \sqrt{3} \approx 1.7\); \(\frac{\pi}{2} \to 2\).
Shape: This is a circle of radius 1 centered at rectangular \((0,1)\).
Circle: diameter 2 on y-axis
Reference
Standard Polar Curve Families
Family Equation Forms Key Characteristics
Circle \(r = a\cos\theta\) or \(r = a\sin\theta\) Radius \(|a|/2\); center sits on an axis.
Cardioid \(r = a \pm b\cos\theta\) (or \(\sin\theta\)) Heart-shaped; passes through the pole (\(a/b = 1\)).
Limaçon (Dimpled) \(r = a \pm b\cos\theta\) (or \(\sin\theta\)) Dimpled boundary, no loop (\(1 < a/b < 2\)).
Limaçon (Inner Loop) \(r = a \pm b\cos\theta\) (or \(\sin\theta\)) Loops through the pole into \(r < 0\) region (\(a < b\)).
Lemniscate \(r^2 = a^2\cos 2\theta\) or \(a^2\sin 2\theta\) Figure-eight shape; maximum distance \(|a|\).
Rose Curve \(r = a\cos n\theta\) or \(a\sin n\theta\) \(n\) odd \(\to n\) petals; \(n\) even \(\to 2n\) petals.
Example 3
Graphing Circle \(r = 4\cos\theta\)
Identify and sketch the graph of \(r = 4\cos\theta\). Find zeros and maximum. OpenStax
Identify Features
Symmetry: About polar axis (since \(\cos(-\theta) = \cos\theta\)).
Zeros: \(\cos\theta = 0 \;\Longrightarrow\; \theta = \frac{\pi}{2}, \frac{3\pi}{2}\).
Maximum: Max \(|r| = 4\) at \(\theta = 0, \pi\).
Resulting Curve
Circle: Center \((2, 0)\) rectangular, radius 2.
Traced completely as \(\theta\) ranges from \(0\) to \(\pi\).
Circle: diameter 4 on x-axis
Example 4
Cardioid \(r = 2 + 2\cos\theta\)
Identify and sketch \(r = 2 + 2\cos\theta\). OpenStax
Identify Type
\(a=2, b=2 \;\Longrightarrow\; \frac{a}{b} = 1 \;\Longrightarrow\) cardioid (heart-shaped).
Symmetry: Symmetric about polar axis (x-axis).
Key Points
Max \(r = 4\) at \(\theta = 0\). Zeros at \(\theta = \pi\) (enters pole).
Values: \(\theta = \frac{\pi}{3} \to r=3\); \(\frac{\pi}{2} \to r=2\); \(\frac{2\pi}{3} \to r=1\).
Cardioid: symmetric about x-axis
Example 5
Dimpled Limaçon \(r = 3 + 2\cos\theta\)
Identify and sketch the curve \(r = 3 + 2\cos\theta\). OpenStax
Check Ratio
\(a=3, b=2 \;\Longrightarrow\; \frac{a}{b} = 1.5\). Since \(1 < 1.5 < 2\), this is a dimpled limaçon.
Analysis
Max \(r = 5\) at \(\theta = 0\). Min \(r = 1\) at \(\theta = \pi\).
Since \(r > 0\) for all \(\theta\), it never crosses the pole (no inner loop, just a dimple at \(\theta=\pi\)).
Dimpled Limaçon: dimple at left
Example 6
Inner-Loop Limaçon \(r = 1 + 2\cos\theta\)
Identify and sketch \(r = 1 + 2\cos\theta\). Note where \(r < 0\). OpenStax
Identify Loop
\(a=1, b=2 \;\Longrightarrow\; a < b \;\Longrightarrow\) limaçon with inner loop.
Zeros: \(1 + 2\cos\theta = 0 \;\Longrightarrow\; \theta = \frac{2\pi}{3}, \frac{4\pi}{3}\).
Inner Loop Analysis
For \(\theta \in \left(\frac{2\pi}{3}, \frac{4\pi}{3}\right)\), \(r < 0\), which traces the inner loop.
Max loop length is \(|1+2\cos\pi| = 1\) (pointing in positive x-direction).
Inner Loop Limaçon
Example 7
Lemniscate \(r^2 = 4\cos 2\theta\)
Identify and sketch the curve \(r^2 = 4\cos 2\theta\). Find zeros and max \(|r|\). OpenStax
Identify Curve
Form \(r^2 = a^2\cos 2\theta \;\Longrightarrow\) lemniscate (figure-eight).
Zeros: \(\cos 2\theta = 0 \;\Longrightarrow\; 2\theta = \frac{\pi}{2} \;\Longrightarrow\; \theta = \frac{\pi}{4}\).
Domain & Max Value
Max \(|r| = 2\) at \(\theta = 0, \pi\).
Undefined where \(\cos 2\theta < 0\) (e.g. \(\theta \in (\frac{\pi}{4}, \frac{3\pi}{4})\)). The graph has two symmetric loops along the polar axis.
Lemniscate: figure-eight
Example 8
Rose Curve \(r = 2\cos 4\theta\)
Identify the family and state the petal count and length for \(r = 2\cos 4\theta\). OpenStax
Petal Count Rule
Form \(r = a\cos n\theta\) with \(n=4\) (even).
Since \(n\) is even, the rose curve has \(2n = 2(4) = \mathbf{8}\) petals.
Petal length is \(|a| = 2\).
Zeros & Symmetry
Zeros: \(\cos 4\theta = 0 \;\Longrightarrow\; 4\theta = \frac{\pi}{2} + k\pi \;\Longrightarrow\; \theta = \frac{\pi}{8} + \frac{k\pi}{4}\).
The first petal is centered at \(\theta = 0\) (on polar axis).
8-Petal Rose Curve
Honors Content
Archimedes' Spiral
Spiral of Archimedes: \(r = \theta \quad (\theta \ge 0)\)
The spiral passes through the pole at \(\theta = 0\) and moves steadily outward as the angle increases.

T-Table: \[\theta = 0 \to r = 0 \qquad \theta = \frac{\pi}{2} \approx 1.57 \qquad \theta = \pi \approx 3.14\] \[\theta = \frac{3\pi}{2} \approx 4.71 \qquad \theta = 2\pi \approx 6.28\] There is no algebraic symmetry because \(-\theta \ne \theta\).
