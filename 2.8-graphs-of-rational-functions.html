2.8 Graphs of Rational Functions
Objective: Sketch any rational function by hand: find the asymptotes (vertical, horizontal, slant), plot the intercepts and a few test points, and connect with smooth curves on each side of the vertical asymptotes.
graphing procedure vertical asymptotes slant asymptotes test points smooth curves
Reference
Guidelines for Graphing a Rational Function
Every rational graph follows the same four-step recipe. The asymptotes are scaffolding; the test points decide the shape between them.
1 · Find & Sketch Asymptotes
Factor numerator and denominator. Cancel common factors to identify holes, then read VA, HA, or slant asymptote from what remains. Draw each asymptote as a dashed line.
2 · Find & Plot Intercepts
\(x\)-intercepts come from zeros of the (simplified) numerator. The \(y\)-intercept is \(f(0)\) — only if \(x = 0\) is in the domain.
3 · Plot Test Points
Plot at least one point between and one point beyond each \(x\)-intercept and each vertical asymptote — the curve's behavior in each region is uniquely determined by sign and asymptote.
4 · Connect Smoothly
Connect the points with smooth curves, respecting the asymptotes (the curve approaches but never crosses VAs; it can cross HAs in the middle but levels off at the ends).
Common Mistake
Don't connect across a vertical asymptote. The graph splits into separate branches at each VA — sketch each branch independently using the test points and the asymptote.
Example 1
Sketch \(g(x) = \dfrac{3}{x - 2}\)
Sketch the graph by hand.
Step 1 — Asymptotes
Set the denominator to zero: \(x - 2 = 0 \Rightarrow x = 2\). Degree of numerator (\(0\)) is less than the denominator (\(1\)).
\[\text{VA: } x = 2, \quad \text{HA: } y = 0\]
Step 2 — Intercepts
Numerator is the constant \(3\), never zero ⇒ no \(x\)-intercept.
\[y\text{-int: } g(0) = \dfrac{3}{-2} = -\dfrac{3}{2} \;\Rightarrow\; \left(0,\, -\tfrac{3}{2}\right)\]
Step 3 — Test points around \(x = 2\)
\[g(1) = \dfrac{3}{-1} = -3, \qquad g(3) = \dfrac{3}{1} = 3, \qquad g(5) = \dfrac{3}{3} = 1\]
Step 4 — Shape
This is the parent \(\tfrac{1}{x}\) shifted right \(2\) and vertically stretched by \(3\). Two branches: one below-left of \((2, 0)\), one above-right.
Sketch
Example 2
Sketch \(f(x) = \dfrac{x^2}{x^2 - x - 2}\)
Sketch the graph by hand.
Step 1 — Factor and find asymptotes
\[f(x) = \dfrac{x^2}{(x - 2)(x + 1)}\]
No common factors ⇒ no holes. Two VAs and a horizontal asymptote from equal degrees.
\[\text{VA: } x = 2,\; x = -1, \quad \text{HA: } y = 1\]
Step 2 — Intercepts
\[x\text{-int: } x^2 = 0 \Rightarrow x = 0\;(\text{mult } 2) \;\Rightarrow\; (0,\, 0)\]
\[y\text{-int: } f(0) = 0 \;\Rightarrow\; (0,\, 0)\]
Step 3 — Test points in each region
\[f(-2) = \dfrac{4}{4} = 1, \qquad f(-3) = \dfrac{9}{10}, \qquad f(1) = -\tfrac{1}{2}, \qquad f(3) = \dfrac{9}{4}\]
Step 4 — Three branches
The two VAs split the plane into three vertical strips. Use the sign of \(f\) and the asymptote behavior in each strip to sketch the branch shape.

Example 3
Sketch \(f(x) = \dfrac{x^2 - 9}{x^2 - 2x - 3}\)
Sketch the graph by hand.
Step 1 — Factor everything
\[f(x) = \dfrac{(x - 3)(x + 3)}{(x - 3)(x + 1)}\]
Step 2 — Cancel for the hole
\((x - 3)\) cancels ⇒ hole at \(x = 3\). Hole \(y\)-value: \(\dfrac{3 + 3}{3 + 1} = \dfrac{3}{2}\). Simplified: \(\dfrac{x + 3}{x + 1}\).
Step 3 — VA and HA
\[\text{VA: } x = -1, \quad \text{HA: } y = 1\]
Step 4 — Intercepts
\[x\text{-int: } x + 3 = 0 \Rightarrow x = -3 \;\Rightarrow\; (-3,\, 0)\]
\[y\text{-int: } f(0) = \dfrac{3}{1} = 3 \;\Rightarrow\; (0,\, 3)\]
Step 5 — Test points
\[f(1) = \dfrac{4}{2} = 2, \qquad f(-2) = \dfrac{1}{-1} = -1, \qquad f(-5) = \dfrac{-2}{-4} = \dfrac{1}{2}\]
Sketch (don't forget the open circle at the hole)
Concept
Slant (Oblique) Asymptotes
When the degree of the numerator is exactly one more than the denominator's, the long-run behavior is a tilted line — a slant asymptote — instead of a flat one.
condition
\[\deg(\text{num}) = \deg(\text{denom}) + 1\]
how to find
Long-divide the numerator by the denominator. The ________quotient (ignore the remainder) is the slant asymptote.
  • A slant asymptote replaces — does not coexist with — a ________horizontal asymptote. A rational function has at most one of the two.
  • If the numerator degree exceeds the denominator's by more than 1, the long-run behavior is a polynomial of higher degree — there is ________no slant asymptote (but the quotient still describes the end behavior).
Example
For \(\dfrac{x^2 + x}{x - 1}\), long division gives \(x + 2\) with remainder \(2\). Slant asymptote: \(y = x + 2\).
Drawing It
Sketch the slant line as a dashed line, just like a horizontal asymptote. The graph hugs this line as \(x \to \pm\infty\).
Synthetic Shortcut
If the divisor is linear, synthetic division gives the quotient quickly. The bottom row reads off the slant-asymptote coefficients.
Example 4
Sketch \(f(x) = \dfrac{x^2 + x}{x - 1}\)
Sketch the graph by hand.
Step 1 — Find the slant asymptote
Degree of numerator (\(2\)) is one more than denominator (\(1\)) ⇒ slant asymptote. Long divide:
\[\dfrac{x^2 + x}{x - 1} = x + 2 + \dfrac{2}{x - 1}\]
\[\text{Slant asymptote: } y = x + 2\]
Step 2 — Vertical asymptote
\[x - 1 = 0 \;\Rightarrow\; x = 1\]
Step 3 — Intercepts
\[x\text{-int: } x^2 + x = x(x + 1) = 0 \Rightarrow x = 0,\; -1 \;\Rightarrow\; (0,\,0),\; (-1,\,0)\]
\[y\text{-int: } f(0) = 0 \Rightarrow (0,\,0)\]
Step 4 — Test points around the VA
\[f(2) = \dfrac{6}{1} = 6, \qquad f(0.5) = \dfrac{0.75}{-0.5} = -\tfrac{3}{2}, \qquad f(-2) = \dfrac{2}{-3} = -\tfrac{2}{3}\]
Step 5 — Behavior near each asymptote
Right of \(x = 1\), the curve rises toward the slant line from below at large \(x\). Left of \(x = 1\), the curve approaches \(y = x + 2\) from above at \(x \to -\infty\).

Example 5
Sketch \(y = \dfrac{x^3}{x^2 - 4}\)
Sketch the graph by hand.
Step 1 — Factor & vertical asymptotes
\[y = \dfrac{x^3}{(x - 2)(x + 2)}\]
\[\text{VAs: } x = 2,\; x = -2\]
Step 2 — Slant asymptote via long division
\[\dfrac{x^3}{x^2 - 4} = x + \dfrac{4x}{x^2 - 4}\]
\[\text{Slant asymptote: } y = x\]
Step 3 — Intercepts
\[x\text{-int: } x^3 = 0 \Rightarrow x = 0 \;(\text{mult } 3) \;\Rightarrow\; (0,\, 0)\]
\[y\text{-int: } y(0) = 0 \;\Rightarrow\; (0,\, 0)\]
Step 4 — Test points
\[y(1) = \dfrac{1}{-3} = -\tfrac{1}{3}, \qquad y(-1) = \dfrac{-1}{-3} = \tfrac{1}{3}\]
\[y(3) = \dfrac{27}{5} = 5.4, \qquad y(-3) = -\tfrac{27}{5} = -5.4\]
Step 5 — Three branches
The two VAs split the graph into three regions. In each region the curve approaches the slant \(y = x\) at the ends and one of the VAs in the middle. The function is odd, so the graph is symmetric about the origin.

Review
Can You Now...?
Use this checklist to confirm you can sketch any rational function by hand.
Follow the Four-Step Recipe
Asymptotes, intercepts, test points, smooth connections — every rational graph follows the same procedure.
Choose the Right Asymptote
Compare degrees: lower numerator ⇒ \(y = 0\). Equal ⇒ leading-coefficient ratio. One more numerator ⇒ slant. More than one ⇒ polynomial end behavior.
Show Holes as Open Dots
A cancelled factor is a hole — a single missing point. Mark it with an open circle on the graph and exclude its \(x\)-value from the domain.
Sketch Each Branch Independently
Vertical asymptotes split the plane into separate vertical strips. Sketch the curve in each strip independently — never connect through a VA.
Use Test Points to Resolve Ambiguity
When the shape isn't obvious, plug in a few \(x\)-values to anchor each branch. Two well-chosen points usually settle it.
Verify End Behavior
As \(x \to \pm\infty\), the graph must approach its horizontal or slant asymptote. If your sketch doesn't, recompute.
A rational graph is the meeting place of two stories: the asymptote scaffolding tells you where the curve is forbidden and how it behaves at the ends; the intercepts and test points tell you exactly which way each branch turns.
