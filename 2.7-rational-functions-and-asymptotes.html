2.7 Rational Functions and Asymptotes
Objective: Identify the domain, vertical and horizontal asymptotes, holes, and intercepts of any rational function by factoring its numerator and denominator and comparing degrees.
rational function vertical asymptote horizontal asymptote hole degree comparison
Reference
What is a Rational Function?
A rational function is a quotient of two polynomials. The denominator cannot equal zero — the points where it does are excluded from the domain.
simple form
\[y = \dfrac{a}{x - h} + k\]
standard form
\[f(x) = \dfrac{g(x)}{h(x)}\quad\text{with}\quad h(x) \neq 0\]
Parent Function
\(f(x) = \dfrac{1}{x},\quad x \neq 0\)
Properties of \(f(x) = \tfrac{1}{x}\)
  • Type of graph: hyperbola
  • Domain: \(\{x \in \mathbb{R} : x \neq 0\}\)
  • Range: \(\{y \in \mathbb{R} : y \neq 0\}\)
  • Decreasing on \((-\infty, 0)\) and \((0, \infty)\)
  • Odd function: \(f(-x) = -f(x)\)
  • Symmetric about the origin
Concept
A Gallery of Rational Functions
Different combinations of numerator and denominator produce wildly different shapes. Each shape is decoded by the same recipe — factor, then compare.
\(y = \dfrac{3x}{x + 1}\)
\(y = \dfrac{x}{(x + 1)(x - 2)}\)
\(y = \dfrac{3}{x^2 + 1}\)
\(y = \dfrac{3}{(x + 1)^2}\)
Linear / Linear
Single VA where the denominator is zero. Horizontal asymptote determined by leading-coefficient ratio.
Multiple Roots in Denom
Multiple distinct denominator roots create multiple vertical asymptotes; the graph splits into separate branches.
No Real Roots in Denom
If the denominator is never zero (like \(x^2 + 1\)), there are no vertical asymptotes — the graph is everywhere continuous.
Reference
Vertical and Horizontal Asymptotes — Definitions
  • If \(f(x) \to \pm\infty\) as \(x \to a\) from the left or right, then \(x = a\) is a ________________vertical asymptote.
  • If \(f(x) \to a\) as \(x \to \pm\infty\), then the line \(y = a\) is a ________________horizontal asymptote.
Vertical Asymptotes
A vertical line the graph approaches but never touches as inputs get close to a forbidden value (a zero of the denominator that doesn't cancel).
Horizontal Asymptotes
A horizontal line the graph levels off toward as \(x\) heads to \(\pm\infty\). Describes the function's long-run behavior.
Difference From Crossings
A graph can cross its horizontal asymptote in the middle and still approach it at the ends — the rule is only about the limits at infinity.

Asymptotes & Holes — The Rules
Interactive Explorer ↗
Compare degrees and look for shared factors. Every feature follows from one of these four rules.
FeatureConditionHow to Find
Vertical Asymptote Values that make ________only the denominator zero Set the denominator equal to \(0\), solve.
Horizontal Asymptote deg(num) ____\(<\) deg(denom) \(y = 0\)
deg(num) ____\(=\) deg(denom) \(y = \dfrac{\text{leading coef. of num}}{\text{leading coef. of denom}}\)
Hole Common factors in ________both numerator and denominator Set the common factor equal to \(0\), solve.
Slant Asymptote deg(num) ____\(>\) deg(denom) (by exactly 1) Synthetic / long division; the quotient (ignoring remainder) is the slant asymptote.
Order of operations matters: factor first, cancel common factors to expose holes, then read off VAs from what remains. Compare degrees of the original rational function for HAs.
Example 1
Domain and Behavior of \(f(x) = \dfrac{1}{x - 1}\)
Find the domain of \(f\) and discuss the behavior of \(f\) near any excluded \(x\)-values.
Step 1 — Domain
\[x - 1 \neq 0 \;\Longrightarrow\; x \neq 1\]
\[\text{Domain: } (-\infty,\, 1) \cup (1,\, \infty)\]
Step 2 — Approach from the left (\(x \to 1^-\))
\(x\)0.90.990.999\(\to 1^-\)
\(f(x)\)\(-10\)\(-100\)\(-1000\)\(\to -\infty\)
Step 3 — Approach from the right (\(x \to 1^+\))
\(x\)\(\to 1^+\)1.0011.011.1
\(f(x)\)\(\to +\infty\)\(1000\)\(100\)\(10\)
Picture & Conclusion
The graph has a vertical asymptote at \(x = 1\) and a horizontal asymptote at \(y = 0\).
Example 2
Find the Asymptotes and/or Holes
(a) \(f(x) = \dfrac{1}{2x}\)
(b) \(f(x) = \dfrac{2x^2}{x^2 - 1}\)
(a) Analyze \(\dfrac{1}{2x}\)
Set denominator \(2x = 0 \Rightarrow x = 0\). Degree of numerator (\(0\)) is less than degree of denominator (\(1\)).
(a) Result
\[\text{VA: } x = 0, \quad \text{HA: } y = 0, \quad \text{no hole}\]
(b) Analyze \(\dfrac{2x^2}{x^2 - 1}\)
\[\dfrac{2x^2}{(x - 1)(x + 1)}\]
Denominator zeros at \(x = \pm 1\). No common factors with numerator. Degrees are equal: \(2 = 2\). HA = ratio of leading coefficients = \(2/1\).
(b) Result
\[\text{VA: } x = 1,\, x = -1, \quad \text{HA: } y = 2, \quad \text{no hole}\]
Example 3
Find the Asymptotes and/or Holes
(a) \(f(x) = \dfrac{3x}{2x^2 + 1}\)
(b) \(y = \dfrac{(x + 4)(x - 2)}{(x - 3)(x + 4)}\)
(a) Analyze \(\dfrac{3x}{2x^2 + 1}\)
Denominator: \(2x^2 + 1 \geq 1 > 0\) for every real \(x\). Denominator is never zero ⇒ no VA.
Degree of numerator (\(1\)) is less than degree of denominator (\(2\)).
(a) Result
\[\text{VA: none}, \quad \text{HA: } y = 0, \quad \text{no hole}\]
(b) Factor and look for common factors
\[\dfrac{(x + 4)(x - 2)}{(x - 3)(x + 4)}\]
\((x + 4)\) appears in both — that's a hole at \(x = -4\). After cancellation: \(\dfrac{x - 2}{x - 3}\).
(b) Read off VA and HA from the simplified form
VA at \(x = 3\). Degrees equal (\(1 = 1\)), HA = leading coef ratio = \(1/1 = 1\).
(b) Result
\[\text{VA: } x = 3, \quad \text{HA: } y = 1, \quad \text{Hole at } \left(-4,\, \tfrac{6}{7}\right)\]
Example 4
Full Analysis: \(y = \dfrac{x^2 + x - 6}{x^2 - x - 2}\)
(a) Identify asymptotes and/or holes. (b) Write the domain. (c) Find the intercepts.
Step 1 — Factor numerator and denominator
\[\dfrac{(x + 3)(x - 2)}{(x - 2)(x + 1)}\]
Step 2 — Cancel common factor → hole
\((x - 2)\) cancels ⇒ hole at \(x = 2\). Hole \(y\)-value: \(\dfrac{2 + 3}{2 + 1} = \dfrac{5}{3}\). Simplified form: \(\dfrac{x + 3}{x + 1}\).
Step 3 — Asymptotes from the simplified form
VA: \(x + 1 = 0 \Rightarrow x = -1\). HA: degrees equal, ratio of leading coefficients \(= 1/1 = 1 \Rightarrow y = 1\).
Step 4 — Domain
Excluded \(x\)-values are the zeros of the original denominator: \(x = 2\) (hole) and \(x = -1\) (VA).
\[\text{Domain: } x \neq 2,\; x \neq -1\]
Step 5 — Intercepts
\[x\text{-int: } x + 3 = 0 \Rightarrow x = -3 \;\;\Rightarrow\;\; (-3,\, 0)\]
\[y\text{-int: } y(0) = \dfrac{0 + 3}{0 + 1} = 3 \;\;\Rightarrow\;\; (0,\, 3)\]
Picture (VA: \(x=-1\), HA: \(y=1\), hole at \((2, \tfrac{5}{3})\))
Example 5
\(f(x) = \dfrac{3x^3 + 7x^2 + 2}{-2x^3 + 16}\)
(a) Identify asymptotes and/or holes. (b) Write the domain.
Step 1 — Factor the denominator
\[-2x^3 + 16 = -2(x^3 - 8) = -2(x - 2)(x^2 + 2x + 4)\]
The quadratic \(x^2 + 2x + 4\) has discriminant \(4 - 16 = -12 < 0\), so it contributes no real zeros.
Step 2 — Test for a hole at \(x = 2\)
\[3(2)^3 + 7(2)^2 + 2 = 24 + 28 + 2 = 54 \neq 0\]
Numerator is non-zero at \(x = 2\), so \((x - 2)\) is not a common factor — no hole.
Step 3 — Vertical asymptote
\[x = 2\]
Step 4 — Horizontal asymptote
Degrees equal: numerator and denominator both have degree \(3\). HA = ratio of leading coefficients.
\[y = \dfrac{3}{-2} = -\dfrac{3}{2}\]
Result
\[\text{VA: } x = 2, \quad \text{HA: } y = -\tfrac{3}{2}, \quad \text{no hole}\]
\[\text{Domain: } x \neq 2\]
Example 6
\(f(x) = \dfrac{x^2 - 16}{x^2 + 8x}\)
(a) Identify asymptotes and/or holes. (b) Write the domain. (c) Find the intercepts.
Step 1 — Factor
\[\dfrac{(x - 4)(x + 4)}{x(x + 8)}\]
Zeros of numerator: \(\pm 4\). Zeros of denominator: \(0,\, -8\). No common factors — no holes.
Step 2 — Vertical asymptotes
\[x = 0 \quad\text{and}\quad x = -8\]
Step 3 — Horizontal asymptote
Degrees equal (\(2 = 2\)). HA = \(1/1 = 1\).
\[y = 1\]
Step 4 — Domain
\[x \neq 0,\quad x \neq -8\]
Step 5 — Intercepts
\[x\text{-int: } x^2 - 16 = 0 \Rightarrow x = \pm 4 \;\;\Rightarrow\;\; (-4,\,0),\; (4,\,0)\]
\[y\text{-int: } f(0) \text{ is undefined } \Rightarrow \text{no }y\text{-intercept}\]
Picture (VAs at \(x=-8\) and \(x=0\); HA at \(y=1\); \(x\)-ints at \(\pm 4\))

Review
Can You Now...?
Use this checklist to confirm you can decode any rational function.
Factor First
Factor numerator and denominator before doing anything else. The features are hidden inside the factors.
Spot Holes
Common factors in numerator and denominator produce holes — not vertical asymptotes. Set the cancelled factor to zero for the \(x\)-coordinate, then plug into the simplified form for the \(y\)-coordinate.
Find Vertical Asymptotes
Set the remaining denominator (after cancellation) equal to zero. Those \(x\)-values are the VAs.
Compare Degrees for HA
Lower numerator degree ⇒ \(y = 0\). Equal degrees ⇒ \(y = \) ratio of leading coefficients. Greater numerator degree ⇒ slant asymptote (next section).
Build the Domain
Exclude every \(x\)-value that makes the original denominator zero — both VAs and holes — even after cancellation.
Find Intercepts
\(x\)-intercepts come from zeros of the numerator (after cancelling holes). The \(y\)-intercept is \(f(0)\), provided \(0\) is in the domain.
A rational function tells its full story in factored form: cancellations are holes, leftover denominator zeros are vertical asymptotes, leftover numerator zeros are \(x\)-intercepts, and the degree comparison decides the horizontal asymptote.
