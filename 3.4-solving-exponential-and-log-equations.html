Unit 3 · Exponential & Logarithmic Functions
3.4 Solving Exponential & Logarithmic Equations
Objective: Solve exponential and logarithmic equations using the one-to-one property, inverse properties, log rules, and substitution. Check for extraneous solutions.
isolate the exponential take the log condense both sides substitution \(u = b^x\) check the domain
Strategy
Three Recurring Moves
Exponential Equations
  1. Isolate the exponential expression.
  2. Take a log (\(\ln\) if base is \(e\), otherwise common log or \(\log_b\)).
  3. Use \(\log_b b^{x} = x\) to drop the exponent.
  4. Solve linearly.
Logarithmic Equations
  1. Use log rules to write each side as a single log.
  2. Apply the one-to-one property (drop the logs) or exponentiate both sides.
  3. Solve, then check: discard any solution that makes an argument \(\le 0\).
When variables appear in two exponents
Either substitute \(u = b^{x}\) (turns it into a polynomial in \(u\)), or take \(\ln\) of both sides and distribute.
Example 1
Solve \(4e^{2x} - 3 = 2\)
Round to the nearest thousandth.
Isolate the exponential
\[4e^{2x} = 5 \;\Longrightarrow\; e^{2x} = \tfrac{5}{4}\]
Take \(\ln\) of both sides
\[\ln e^{2x} = \ln \tfrac{5}{4} \;\Longrightarrow\; 2x = \ln \tfrac{5}{4}\]
Solve for \(x\)
\[x = \tfrac{1}{2} \ln \tfrac{5}{4}\]
Approximate
\[x \approx \tfrac{1}{2}(0.22314) \approx 0.112\]
Example 2
Solve \(2\!\left(3^{\,2t - 5}\right) - 4 = 11\)
Round to the nearest thousandth.
Isolate the exponential
\[2\!\left(3^{\,2t-5}\right) = 15 \;\Longrightarrow\; 3^{\,2t-5} = \tfrac{15}{2}\]
Take \(\ln\) (or use change of base)
\[(2t - 5)\ln 3 = \ln \tfrac{15}{2} \;\Longrightarrow\; 2t - 5 = \dfrac{\ln 7.5}{\ln 3}\]
Solve for \(t\)
\[2t = 5 + \dfrac{\ln 7.5}{\ln 3} \;\Longrightarrow\; t = \tfrac{1}{2}\!\left(5 + \dfrac{\ln 7.5}{\ln 3}\right)\]
Approximate
\[t \approx \tfrac{1}{2}(5 + 1.83399) \approx 3.417\]
Example 3
Solve \(5^{\,2x + 1} = 6^{\,x - 2}\)
Variables in two different bases — take logs of both sides.
Take \(\ln\) of both sides
\[(2x + 1)\ln 5 = (x - 2)\ln 6\]
Distribute
\[2x \ln 5 + \ln 5 = x \ln 6 - 2 \ln 6\]
Collect \(x\)-terms on one side
\[x(2\ln 5 - \ln 6) = -2\ln 6 - \ln 5\]
Solve for \(x\)
\[x = \dfrac{-2\ln 6 - \ln 5}{2\ln 5 - \ln 6}\]
Approximate
\[x \approx \dfrac{-5.193}{1.427} \approx -3.639\]
Example 4
Solve \(4(2^{x}) + 8(2^{-x}) = 33\)
Variable appears as \(2^{x}\) and \(2^{-x}\) — use the substitution \(u = 2^{x}\).
Substitute \(u = 2^{x}\) (so \(2^{-x} = 1/u\))
\[4u + \dfrac{8}{u} = 33\]
Multiply by \(u\) and rearrange
\[4u^{2} - 33u + 8 = 0\]
Quadratic formula (or factor: \((4u - 1)(u - 8) = 0\))
\[u = 8 \quad \text{or} \quad u = \tfrac{1}{4}\]
Undo the substitution
\[2^{x} = 8 = 2^{3} \;\Rightarrow\; x = 3 \qquad \text{or} \qquad 2^{x} = \tfrac{1}{4} = 2^{-2} \;\Rightarrow\; x = -2\]
Answer
\[x = -2 \quad \text{or} \quad x = 3\]
Example 5
Solve \(\dfrac{5^{x} - 5^{-x}}{2} = 3\)
Same substitution trick as before: let \(u = 5^{x}\).
Clear the denominator
\[5^{x} - 5^{-x} = 6\]
Substitute \(u = 5^{x}\) and multiply by \(u\)
\[u - \tfrac{1}{u} = 6 \;\Longrightarrow\; u^{2} - 6u - 1 = 0\]
Quadratic formula
\[u = \dfrac{6 \pm \sqrt{36 + 4}}{2} = 3 \pm \sqrt{10}\]
Keep only \(u > 0\) (since \(u = 5^{x} > 0\))
\[5^{x} = 3 + \sqrt{10} \;\Longrightarrow\; x = \dfrac{\ln(3 + \sqrt{10})}{\ln 5}\]
Approximate
\[x \approx \dfrac{\ln 6.162}{\ln 5} \approx \dfrac{1.818}{1.609} \approx 1.130\]
Example 6
Solve \(5 + 2\ln x = 4\)
Round to the nearest thousandth.
Isolate the log
\[2 \ln x = -1 \;\Longrightarrow\; \ln x = -\tfrac{1}{2}\]
Exponentiate (apply \(e^{(\cdot)}\) to both sides)
\[e^{\ln x} = e^{-1/2} \;\Longrightarrow\; x = e^{-1/2}\]
Approximate
\[x \approx 0.607\]
Check the domain
\(x \approx 0.607 > 0\) ✓ — valid input for \(\ln x\).
Example 7
Solve \(\log_{5}(2x + 3) = \log_{5} 11 + \log_{5} 5\)
Same base on both sides — condense, then use the one-to-one property.
Condense the right side (product property)
\[\log_{5}(2x + 3) = \log_{5}(11 \cdot 5) = \log_{5} 55\]
Drop the logs (one-to-one)
\[2x + 3 = 55\]
Solve for \(x\)
\[2x = 52 \;\Longrightarrow\; x = 26\]
Check & answer
Argument \(2(26) + 3 = 55 > 0\) ✓   \(\Rightarrow\) \(x = 26\)
Example 8
Solve \(\log_{2} x + \log_{2}(x + 2) = 3\)
Watch for an extraneous solution at the end.
Condense the left side
\[\log_{2}\!\left[x(x + 2)\right] = 3\]
Rewrite in exponential form
\[x(x + 2) = 2^{3} = 8\]
Solve the quadratic
\[x^{2} + 2x - 8 = 0 \;\Longrightarrow\; (x + 4)(x - 2) = 0\]
Check both candidates
\(x = -4\): \(\log_{2}(-4)\) is undefined  extraneous.   \(x = 2\): both arguments positive ✓
Answer
\[x = 2\]
Example 9
Solve \(\log \sqrt[3]{x} = \sqrt{\log x}\)
Challenge — combine the power property with a substitution.
Bring the radical down using the power property
\[\tfrac{1}{3} \log x = \sqrt{\log x}\]
Substitute \(u = \sqrt{\log x}\), so \(u^{2} = \log x\)
\[\tfrac{1}{3} u^{2} = u \;\Longrightarrow\; u^{2} - 3u = 0 \;\Longrightarrow\; u(u - 3) = 0\]
Back-substitute
\[u = 0 \Rightarrow \log x = 0 \Rightarrow x = 1 \qquad u = 3 \Rightarrow \log x = 9 \Rightarrow x = 10^{9}\]
Check each candidate
\(x = 1\): \(\log 1 = 0\), \(\sqrt{0} = 0\) ✓    \(x = 10^{9}\): \(\tfrac{1}{3}(9) = 3 = \sqrt{9}\) ✓
Answer
\[x = 1 \quad \text{or} \quad x = 10^{9}\]
