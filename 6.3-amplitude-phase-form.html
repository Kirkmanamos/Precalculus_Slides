Unit 5 · Analytic Trigonometry
5.3 Amplitude-Phase Form
Objective: Rewrite \(a\sin x+b\cos x\) as one shifted sine wave and use that form to simplify equations.
amplitude phase angle sum identity coefficient vector
Reference
The One-Sine-Wave Form
Any expression of the form \(a\sin x+b\cos x\) can be written as \(R\sin(x+\theta)\).
Start with the sum identity
\[ R\sin(x+\theta) =R\sin x\cos\theta+R\cos x\sin\theta \]
Match coefficients
\[ a=R\cos\theta \qquad\text{and}\qquad b=R\sin\theta \]
The sine coefficient is the horizontal component. The cosine coefficient is the vertical component.
Compute the amplitude and phase angle
\[ R=\sqrt{a^2+b^2}, \qquad \theta=\operatorname{atan2}(b,a) \]
Example 1
Rewrite as One Shifted Sine Wave
Rewrite \(\sqrt{3}\sin x-\cos x\) in amplitude-phase form.
Coefficients
\(a=\sqrt{3},\ b=-1\)
Amplitude
\(R=2\)
Phase angle
\(\theta=-\dfrac{\pi}{6}\)
Identify the target form
\[ \sqrt{3}\sin x-\cos x=R\sin(x+\theta) \]
Find the amplitude
\[ R=\sqrt{(\sqrt{3})^2+(-1)^2}=\sqrt{4}=2 \]
Find the phase angle
\[ \cos\theta=\dfrac{\sqrt{3}}{2}, \qquad \sin\theta=-\dfrac{1}{2} \quad\Rightarrow\quad \theta=-\dfrac{\pi}{6} \]
Amplitude-phase form
\[ \sqrt{3}\sin x-\cos x=2\sin\left(x-\dfrac{\pi}{6}\right) \]
Verification
\[ 2\sin\left(x-\dfrac{\pi}{6}\right) =2\left(\sin x\cdot\dfrac{\sqrt{3}}{2}-\cos x\cdot\dfrac{1}{2}\right) =\sqrt{3}\sin x-\cos x \]
Explorer
Changing the Coefficients
Amplitude
Phase angle
Equivalent form
\(a\sin x+b\cos x\) \(R\sin(x+\theta)\)
Original
Shifted sine
Coefficient vector
The point \((a,b)\) has radius \(R\). Its direction angle is \(\theta\), so \(a=R\cos\theta\) and \(b=R\sin\theta\).
Phase shift sign
\(R\sin(x+\theta)=R\sin(x-C)\) where \(C=-\theta\). A negative \(\theta\) means a right shift.
Example 2
A Common Special Case
Rewrite \(\sin x+\cos x\) in amplitude-phase form.
Identify the coefficients
\[ a=1,\qquad b=1 \]
Find the amplitude
\[ R=\sqrt{1^2+1^2}=\sqrt{2} \]
Find the phase angle
\[ \cos\theta=\dfrac{1}{\sqrt{2}}, \qquad \sin\theta=\dfrac{1}{\sqrt{2}} \quad\Rightarrow\quad \theta=\dfrac{\pi}{4} \]
Answer
\[ \sin x+\cos x=\sqrt{2}\sin\left(x+\dfrac{\pi}{4}\right) \]
Example 3
Use Phase Form to Solve
Solve \(\sqrt{3}\sin x-\cos x=1\) on \(0\le x<2\pi\).
Use the form from Example 1
\[ \sqrt{3}\sin x-\cos x=2\sin\left(x-\dfrac{\pi}{6}\right) \]
Set up a simple sine equation
\[ 2\sin\left(x-\dfrac{\pi}{6}\right)=1 \quad\Rightarrow\quad \sin\left(x-\dfrac{\pi}{6}\right)=\dfrac{1}{2} \]
Solve for the shifted angle
\[ x-\dfrac{\pi}{6}=\dfrac{\pi}{6} \quad\text{or}\quad x-\dfrac{\pi}{6}=\dfrac{5\pi}{6} \]
Add the phase shift back
\[ x=\dfrac{\pi}{3} \quad\text{or}\quad x=\pi \]
Solution set
\[ \boxed{x=\dfrac{\pi}{3},\ \pi} \]
Review
What to Remember
Starting expression
\[ a\sin x+b\cos x \]
=
Amplitude-phase form
\[ \sqrt{a^2+b^2}\,\sin(x+\theta) \]
Amplitude
The new amplitude is \(R=\sqrt{a^2+b^2}\), the length of the coefficient vector.
Phase Angle
Choose \(\theta\) so that \(\cos\theta=\dfrac{a}{R}\) and \(\sin\theta=\dfrac{b}{R}\).
Sign Check
In \(R\sin(x+\theta)\), the graph shift is \(C=-\theta\). Check the sign before graphing or solving.
Fast check: expand \(R\sin(x+\theta)\). If the sine and cosine coefficients match the original expression, the form is correct.
