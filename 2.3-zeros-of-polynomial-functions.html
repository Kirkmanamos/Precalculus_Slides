2.3 Zeros of Polynomial Functions
Objective: Translate freely between five names for the same idea (zero, root, factor, \(x\)-intercept, solution), sketch polynomials from factored form, build a polynomial from given zeros, and use the Intermediate Value Theorem to locate hidden zeros.
zeros roots factors intercepts IVT
Vocabulary
Five Names, One Idea
When the polynomial \(f(x)\) equals zero at \(x = a\), five different vocabulary words describe the same fact — depending on what object we're naming.
  • \(x = a\) is a ________zero of the function \(f(x)\).
  • \(x = a\) is a ________solution of the polynomial equation \(f(x) = 0\).
  • \((x - a)\) is a ________factor of the polynomial \(f(x)\).
  • \((a,\, 0)\) is an ________________\(x\)-intercept of the graph of \(f(x)\).
  • Sometimes zeros are called ________roots.
Same Number, Different Object
All five expressions name the value \(a\). The vocabulary just changes when we change what we're talking about — function, equation, polynomial, or graph.
Read the Question Carefully
If the problem says "find the roots," "find the zeros," "find the solutions," or "find the \(x\)-intercepts" — it is asking for the same thing.
Bridge to Factoring
If \(a\) is a zero, then \((x - a)\) divides the polynomial. This connects the algebra of factoring to the geometry of crossing the \(x\)-axis.
Example 1
\(f(x) = x^3 - x^2 - 2x\)
Use the zeros, \(y\)-intercept, and end behavior to sketch the graph.
Step 1 — Factor
\[f(x) = x(x^2 - x - 2) = x(x - 2)(x + 1)\]
Step 2 — Zeros (each multiplicity 1)
\[x = 0,\; x = 2,\; x = -1 \quad(\text{all crossings})\]
Step 3 — \(y\)-intercept and end behavior
\(f(0) = 0\), so the graph passes through the origin. Degree \(3\), leading coefficient \(+1\) ⇒ left ↓, right ↑.
Step 4 — Sign check
Between zeros, pick a test value: \(f(-2) < 0\), \(f(-0.5) > 0\), \(f(1) < 0\), \(f(3) > 0\). The curve weaves across the axis at each zero.
Sketch
Example 2
\(g(x) = x^3 - 2x^2 - 4x + 8\)
Use the zeros, \(y\)-intercept, and end behavior to sketch the graph.
Step 1 — Factor by grouping
\[g(x) = x^2(x - 2) - 4(x - 2) = (x^2 - 4)(x - 2)\]
Step 2 — Finish factoring
\[g(x) = (x - 2)(x + 2)(x - 2) = (x - 2)^2 (x + 2)\]
Step 3 — Zeros and multiplicities
\[x = 2\;\; (\text{mult } 2,\text{ tangent}), \qquad x = -2\;\; (\text{mult } 1,\text{ crosses})\]
Step 4 — \(y\)-intercept
\[g(0) = 8 \quad\Rightarrow\quad (0,\, 8)\]
Step 5 — End behavior
Degree \(3\), leading coefficient \(+1\) ⇒ left ↓, right ↑. The double root at \(x = 2\) means the curve touches and bounces back instead of crossing.
Sketch
Example 3
\(f(x) = -2x^4 - x^3 + 3x^2\)
Use the zeros, \(y\)-intercept, and end behavior to sketch the graph.
Step 1 — Factor out the GCF
\[f(x) = -x^2(2x^2 + x - 3)\]
Step 2 — Factor the trinomial
\[f(x) = -x^2(2x + 3)(x - 1)\]
Step 3 — Zeros and multiplicities
\[x = 0\;\;(\text{mult } 2), \qquad x = -\tfrac{3}{2}, \qquad x = 1\]
Step 4 — End behavior and y-intercept
Degree \(4\), leading coefficient \(-2\) ⇒ both ends ↓. The double zero at the origin means the graph is tangent there. \(f(0) = 0\).
Sketch
Example 4
\(f(x) = x^4 - 12x^2 + 27\)
Use the zeros, \(y\)-intercept, and end behavior to sketch the graph.
Step 1 — Quadratic-type substitution
\[\text{Let } u = x^2:\quad u^2 - 12u + 27 = 0\]
Exponents follow the quadratic-type pattern (4 is twice 2).
Step 2 — Factor and solve for \(u\)
\[(u - 3)(u - 9) = 0 \;\Longrightarrow\; u = 3 \;\text{ or }\; u = 9\]
Step 3 — Back-substitute \(u = x^2\)
\[x^2 = 3 \;\Rightarrow\; x = \pm\sqrt{3}, \qquad x^2 = 9 \;\Rightarrow\; x = \pm 3\]
Step 4 — End behavior and \(y\)-intercept
Degree \(4\), leading coefficient \(+1\) ⇒ both ends ↑. \(f(0) = 27\) — a tall \(y\)-intercept. Four distinct zeros: \(\pm\sqrt{3} \approx \pm 1.73\) and \(\pm 3\).
Sketch
Concept
Build a Polynomial from Its Zeros
Reverse the process. If a polynomial has zeros \(r_1, r_2, \dots, r_n\), then we can construct it as a product of factors.
construction
\[f(x) = a\,(x - r_1)(x - r_2)\cdots(x - r_n)\]
Repeated Zero
A zero \(r\) listed twice contributes a factor \((x - r)^2\). The exponent is the multiplicity.
Fractional Zero
If a zero is \(-\tfrac{1}{2}\), use the factor \((2x + 1)\) instead of \((x + \tfrac{1}{2})\) to keep the coefficients whole.
Conjugate Radicals
If \(2 + \sqrt{3}\) is a zero, then so is \(2 - \sqrt{3}\). Multiplying conjugate factors gives a clean quadratic with rational coefficients.
Leading Coefficient
Without more information, set \(a = 1\). Use \(a\) to scale the function or fix its sign if additional conditions are given (a known point, end behavior, etc.).
Example 5
Zeros: \(-\tfrac{1}{2},\; 3,\; 3\)
Write an equation of a polynomial function with the given zeros.
Step 1 — Turn each zero into a factor
\[\left(x + \tfrac{1}{2}\right),\quad (x - 3),\quad (x - 3)\]
Step 2 — Clear the fraction
\[\left(x + \tfrac{1}{2}\right) \cdot 2 = (2x + 1)\]
Multiplying a single factor by 2 only rescales the polynomial — it doesn't change the zeros.
Step 3 — Multiply out
\[f(x) = (2x + 1)(x - 3)^2\]
Expanded Form
\[f(x) = 2x^3 - 11x^2 + 12x + 9\]
Example 6
Zeros: \(3,\; 2 + \sqrt{3},\; 2 - \sqrt{3}\)
Write an equation of a polynomial function with the given zeros.
Step 1 — Turn each zero into a factor
\[(x - 3),\quad \bigl(x - (2 + \sqrt{3})\bigr),\quad \bigl(x - (2 - \sqrt{3})\bigr)\]
Step 2 — Multiply the conjugate pair first
\[\bigl(x - (2 + \sqrt{3})\bigr)\bigl(x - (2 - \sqrt{3})\bigr) = \bigl((x - 2) - \sqrt{3}\bigr)\bigl((x - 2) + \sqrt{3}\bigr)\]
Group \((x - 2)\) together — the square roots will cancel via the difference-of-squares pattern.
Step 3 — Apply \((A - B)(A + B) = A^2 - B^2\)
\[= (x - 2)^2 - 3 = x^2 - 4x + 1\]
Step 4 — Multiply by the remaining factor
\[f(x) = (x - 3)(x^2 - 4x + 1)\]
Expanded Form
\[f(x) = x^3 - 7x^2 + 13x - 3\]
Reference
Intermediate Value Theorem
A continuous function cannot jump over the \(x\)-axis. If the function is positive at one place and negative at another, it must hit zero somewhere in between.
statement
If \(f\) is a polynomial and ________________\(f(a) < 0\) while ________________\(f(b) > 0\) (or vice versa), then \(f\) has at least one zero between \(a\) and \(b\).
Picture
What It Proves
A zero exists in the interval \((a, b)\). The theorem doesn't tell you where, only that.
When It Fails
IVT needs a sign change. If \(f(a)\) and \(f(b)\) have the same sign, the test is inconclusive — zeros may still exist, but IVT won't find them.
Example 7
Use IVT on \(f(x) = x^4 - 4x + 1\)
Show that there is at least one zero of \(f(x) = x^4 - 4x + 1\) between \(x = 0\) and \(x = 1\).
Step 1 — Evaluate at the endpoints
\[f(0) = 0^4 - 4(0) + 1 = 1\]
\[f(1) = 1^4 - 4(1) + 1 = -2\]
Step 2 — Note the sign change
\(f(0) = 1 > 0\) and \(f(1) = -2 < 0\). The function values have opposite signs.
Step 3 — Cite continuity
\(f\) is a polynomial, so it is continuous everywhere — in particular on \([0, 1]\).
Conclusion (by IVT)
Because \(f\) is continuous on \([0, 1]\) and \(f(0)\) and \(f(1)\) have opposite signs, there exists at least one \(c \in (0, 1)\) with \(f(c) = 0\). \(\square\)

Review
Can You Now...?
Use this checklist to confirm you've mastered the section.
Translate the Vocabulary
Move freely between zero, root, solution, factor, and \(x\)-intercept — five words for the same number.
Sketch From Factors
Factor a polynomial, identify the multiplicity at each zero, anchor the \(y\)-intercept, and use end behavior to sweep the arms.
Recognize Quadratic Type
When you see \(x^4 - 12x^2 + 27\) or similar, substitute \(u = x^2\) and solve the quadratic for \(u\), then back-substitute.
Build From Zeros
Convert each zero into a factor, multiply, and clear fractions or radicals by pairing conjugates.
Pair Conjugate Radicals
Multiply \((x - (p + \sqrt{q}))(x - (p - \sqrt{q})) = (x - p)^2 - q\) — a tidy quadratic with no radicals.
Apply IVT
Check that \(f\) is continuous, find a sign change \(f(a) \cdot f(b) < 0\), and conclude that a zero must live in \((a, b)\).
Every "find the zeros" question is the same: factor, then read each factor as a zero. Every "build the polynomial" question is the reverse: read each zero as a factor, then multiply.
