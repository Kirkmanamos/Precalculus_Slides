A Periodic Function...

...is a function that returns to the same value at regular intervals.

Period

The least amount of space (degrees or radians) the function takes to complete one cycle.

Amplitude

Half the distance between the maximum and minimum values.

General Form

y = a sin(b(x - c)) + d

a = amplitude (vertical stretch/shrink)

b = affects period (horizontal stretch/shrink)

c = phase shift (horizontal translation)

d = vertical translation

Note

We will use:

y = a sin(b(x - c)) + d

phase shift = c

Some resources use:

y = a sin(bx - c) + d

phase shift = c/b

Note

These are not the same:

y = sin(2x - π)

Rewrite as: y = sin(2(x - π/2))

phase shift = π/2
