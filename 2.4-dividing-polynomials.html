2.4 Dividing Polynomials
Objective: Divide polynomials by long division and synthetic division, apply the Remainder and Factor Theorems to evaluate functions and verify factors, and use the Rational Zero Theorem to find rational zeros.
long division synthetic division Remainder Theorem Factor Theorem Rational Zeros
Reference
The Division Algorithm
Polynomial division mirrors the integer division you've known since elementary school. Compare them side by side.
Integer Example
\(250 = 3(83) + 1\)

\[\dfrac{250}{3} = 83 + \dfrac{1}{3}\]
250 is the dividend, 3 the divisor, 83 the quotient, 1 the remainder.
Polynomial Form
\[\text{dividend} = (\text{divisor})(\text{quotient}) + \text{remainder}\] \[\dfrac{\text{dividend}}{\text{divisor}} = \text{quotient} + \dfrac{\text{remainder}}{\text{divisor}}\]
Same Four Names
Dividend, divisor, quotient, remainder. The polynomial division algorithm uses identical vocabulary.
Stop Condition
Long division stops when the remainder has lower degree than the divisor (just like the remainder must be less than the divisor for integers).
Two Methods
Use long division for any divisor. Use synthetic division when the divisor is linear, of the form \((x - c)\) — it's faster and avoids the bookkeeping.
Example 1
Long Division: \((4x^3 + x^2 - 16x - 4) \div (x + 2)\)
Divide the dividend by the divisor using polynomial long division.
Step 1 — Divide leading terms
\[\dfrac{4x^3}{x} = 4x^2\]
Multiply back: \(4x^2(x+2) = 4x^3 + 8x^2\). Subtract from the dividend.
\[(4x^3 + x^2 - 16x - 4) - (4x^3 + 8x^2) = -7x^2 - 16x - 4\]
Step 2 — Repeat on the new dividend
\[\dfrac{-7x^2}{x} = -7x\]
Multiply back: \(-7x(x+2) = -7x^2 - 14x\). Subtract.
\[(-7x^2 - 16x - 4) - (-7x^2 - 14x) = -2x - 4\]
Step 3 — One more time
\[\dfrac{-2x}{x} = -2\]
Multiply back: \(-2(x+2) = -2x - 4\). Subtract.
\[(-2x - 4) - (-2x - 4) = 0\]
Result
\[\text{Quotient} = 4x^2 - 7x - 2, \qquad \text{Remainder} = 0\]
Conclusion
\[\dfrac{4x^3 + x^2 - 16x - 4}{x + 2} = 4x^2 - 7x - 2\]
The remainder is \(0\), so \((x+2)\) divides the dividend evenly — \((x+2)\) is a factor.
Example 2
\((2x^4 + 4x^3 + 3x - 2) \div (x^2 + 2x - 3)\)
Divide a quartic by a quadratic. Watch for the missing \(x^2\) term in the dividend — write \(2x^4 + 4x^3 + 0x^2 + 3x - 2\).
Step 1 — Divide leading terms
\[\dfrac{2x^4}{x^2} = 2x^2\]
Multiply: \(2x^2(x^2 + 2x - 3) = 2x^4 + 4x^3 - 6x^2\). Subtract.
\[(2x^4 + 4x^3 + 0x^2 + 3x - 2) - (2x^4 + 4x^3 - 6x^2) = 6x^2 + 3x - 2\]
Step 2 — Repeat
\[\dfrac{6x^2}{x^2} = 6\]
Multiply: \(6(x^2 + 2x - 3) = 6x^2 + 12x - 18\). Subtract.
\[(6x^2 + 3x - 2) - (6x^2 + 12x - 18) = -9x + 16\]
Step 3 — Stop
The remainder \(-9x + 16\) has degree \(1\), lower than the divisor's degree \(2\). We're done.
Result
\[\dfrac{2x^4 + 4x^3 + 3x - 2}{x^2 + 2x - 3} = 2x^2 + 6 + \dfrac{-9x + 16}{x^2 + 2x - 3}\]
Concept
Synthetic Division — A Shortcut for \((x - c)\)
When the divisor is a linear factor \((x - c)\), the entire long-division algorithm collapses into a tidy three-row table. Only the coefficients matter.
Step 1 — Write Coefficients
List the coefficients of the dividend in descending order of powers. Use 0 for any missing term.
Step 2 — Use \(c\), Not \((x - c)\)
If the divisor is \((x + 2)\), then \(c = -2\). Always flip the sign of the constant in the divisor.
Step 3 — Bring Down, Multiply, Add
Bring down the first coefficient. Multiply by \(c\), write under the next coefficient, add. Repeat across the row.
Reading the Result
The bottom row gives you the coefficients of the quotient (degree one less than the dividend). The last number in the bottom row is the remainder.

Reference
Remainder & Factor Theorems
Remainder Thm.
If a polynomial \(f(x)\) is divided by a linear factor of the form ________\((x - c)\), then the remainder is equal to ________\(f(c)\).
Factor Thm.
If ________\(f(c) = 0\) for a polynomial \(f(x)\), then ________\((x - c)\) is a factor of \(f(x)\).
Two Sides of the Same Coin
The Remainder Theorem lets you evaluate a polynomial via synthetic division. The Factor Theorem lets you verify a factor by plugging in.
Quick Check for Zero
To check whether \(x = c\) is a zero, compute \(f(c)\). If \(f(c) = 0\), then \(c\) is a zero and \((x - c)\) is a factor.
Two Ways to Compute \(f(c)\)
Plug in directly, or run synthetic division and read the remainder. Synthetic division is often faster for high-degree polynomials.

The Rational Zero Theorem
When a polynomial has integer coefficients, every rational zero must come from a short, predictable list. We don't have to guess.
if zero is
\(\dfrac{p}{q}\) (in lowest terms)
then
\(p\) is a factor of the ________________constant term, and \(q\) is a factor of the ________________leading coefficient.
Working Formula
\[\text{possible rational zeros} \;=\; \pm\dfrac{\text{factors of the constant term}}{\text{factors of the leading coefficient}}\]
List, Don't Guess
Write out every \(\pm p/q\) combination, then test with synthetic division. Stop testing once you find a zero — divide and continue with the smaller quotient.
Lowest Terms Matters
Reduce duplicates: \(\pm 1/1\) is just \(\pm 1\). \(\pm 2/2\) is just \(\pm 1\). Otherwise you'll waste time testing the same value twice.
Doesn't Guarantee Zeros
The theorem only tells you which rationals could be zeros. The polynomial may have all irrational zeros — in which case nothing on the list will work.
Example 7
Rational Zeros of \(P(x) = 4x^3 - 4x^2 - x + 1\)
Find all the rational zeros of \(P(x)\).
Step 1 — List the candidates
Constant term \(1\): factors \(\pm 1\). Leading coefficient \(4\): factors \(\pm 1, \pm 2, \pm 4\).
\[\dfrac{p}{q} = \pm 1,\; \pm\tfrac{1}{2},\; \pm\tfrac{1}{4}\]
Step 2 — Test the candidates
1 ✓ -1 1/2 ✓ -1/2 ✓ 1/4 -1/4
\[P(1) = 4 - 4 - 1 + 1 = 0 \;\checkmark\]
Step 3 — Factor by grouping (sanity check)
\[P(x) = 4x^2(x - 1) - 1(x - 1) = (x - 1)(4x^2 - 1)\]
Step 4 — Factor the difference of squares
\[P(x) = (x - 1)(2x - 1)(2x + 1)\]
All Rational Zeros
\[x = 1,\quad x = \tfrac{1}{2},\quad x = -\tfrac{1}{2}\]

Step 1 — Build the candidate list
Constant \(12\): factors \(\pm 1, 2, 3, 4, 6, 12\). Leading \(10\): factors \(\pm 1, 2, 5, 10\). Many candidates; test small integers first.

Review
Can You Now...?
Use this checklist to make sure you can divide, factor, and find zeros of any polynomial.
Use the Division Algorithm
Write any polynomial division as \(\text{dividend} = (\text{divisor})(\text{quotient}) + \text{remainder}\).
Long Division
Always usable. Insert \(0\) for missing terms in the dividend. Stop when the remainder has lower degree than the divisor.
Synthetic Division
Faster, but only works for \((x - c)\) divisors. Use \(c\), not \((x - c)\), and remember the sign flip.
Remainder Theorem
\(f(c)\) equals the remainder when \(f(x)\) is divided by \((x - c)\). Use synthetic division to evaluate a polynomial.
Factor Theorem
If \(f(c) = 0\), then \((x - c)\) is a factor — and conversely. The bridge between zeros and factors.
Rational Zero Theorem
Possible rational zeros = \(\pm\dfrac{\text{factors of constant}}{\text{factors of leading coef.}}\). Test each with synthetic division until you find one.
Every polynomial-zero problem is a hunt: list the rational candidates, test them, divide to peel off each factor, and use the quadratic formula on whatever remains.
