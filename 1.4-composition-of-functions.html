Chapter 1 · Functions
1.4 Composition of Functions
Objective: Combine functions algebraically and determine their domains, evaluate composite functions numerically and algebraically, find composite domains, and decompose functions.
\(f \pm g\) \(f \cdot g\) \(f / g\) \(f \circ g\) composite domains
Reference
Arithmetic Operations on Functions
OpenStax
Definitions
• Sum: \((f + g)(x) = f(x) + g(x)\)
• Difference: \((f - g)(x) = f(x) - g(x)\)
• Product: \((f \cdot g)(x) = f(x) \cdot g(x)\)
• Quotient: \(\left(\dfrac{f}{g}\right)(x) = \dfrac{f(x)}{g(x)}\)
Combined Domain Rules
The domain of \(f \pm g\), \(f \cdot g\), and \(\dfrac{f}{g}\) is the ________________intersection of the domains of \(f\) and \(g\) (where they overlap).
For division, we must also exclude any \(x\) where ________________\(g(x) = 0\).
Step 1 · Intersection
Find the domain of \(f\) and the domain of \(g\) separately, then find their overlapping intervals.
Step 2 · Zero Exclusions
For division, identify where the denominator function equals zero and remove those coordinates from the domain.
Example 1
Find \((f - g)(x)\)
Larson
Given
\(f(x) = 3x^2 + 7x\) and \(g(x) = 2x^2 - x - 1\)
Set up
\[(f - g)(x) = (3x^2 + 7x) - (2x^2 - x - 1)\]
Distribute the negative sign
\[= 3x^2 + 7x - 2x^2 + x + 1\]
Group like terms
\[= (3x^2 - 2x^2) + (7x + x) + 1\]
Answer
\[(f - g)(x) = x^2 + 8x + 1\]
Example 2
Find \(\left(\tfrac{f}{g}\right)(x)\) and state its Domain
OpenStax
Given
\(f(x) = 3x^2 - 2x + 1\) and \(g(x) = x - 4\)
Form the quotient
\[\left(\dfrac{f}{g}\right)(x) = \dfrac{3x^2 - 2x + 1}{x - 4}\]
Domains separately
\(f\) is a polynomial \(\Rightarrow (-\infty,\, \infty)\). \(\;g\) is a polynomial \(\Rightarrow (-\infty,\, \infty)\).
Exclude zeroes of the denominator
Set \(g(x) = 0 \;\Longrightarrow\; x - 4 = 0 \;\Longrightarrow\; x = 4\). This point must be excluded.
Final Domain
\[(-\infty,\, 4) \cup (4,\, \infty) \quad \text{or} \quad \{x \in \mathbb{R} \mid x \neq 4\}\]
Concept
Composition of Functions
OpenStax
Composition Notation Pronunciation
\(f\) composed with \(g\) \((f \circ g)(x) = f(g(x))\) "\(f\) of \(g\) of \(x\)"
\(g\) composed with \(f\) \((g \circ f)(x) = g(f(x))\) "\(g\) of \(f\) of \(x\)"
Crucial Rule: Not Multiplication!
\((f \circ g)(x)\) is **not** \(f(x) \cdot g(x)\). The open circle \(\circ\) represents nesting one function inside another.
Inside Out Strategy
Always evaluate composition from the **inside out**. Find the output of the inner function first, then use that output as the input for the outer function.
Order Matters
Composition is not commutative. In almost all cases, \((f \circ g)(x) \neq (g \circ f)(x)\).
Example 3
Evaluate Composition Numerically
OpenStax
Given
\(f(x) = 2x + 3\) and \(g(x) = x^2 - 1\)
a) Evaluate \(f(g(1))\) · Inner first
\(g(1) = (1)^2 - 1 = 0\)
Feed result into outer function
\(f(g(1)) = f(0) = 2(0) + 3\)
Answer a)
\(f(g(1)) = 3\)
b) Evaluate \(g(f(2))\) · Inner first
\(f(2) = 2(2) + 3 = 7\)
Feed result into outer function
\(g(f(2)) = g(7) = (7)^2 - 1\)
Answer b)
\(g(f(2)) = 48\)
Example 4
Evaluate Composition Algebraically
OpenStax
Given
\(f(x) = x - 3\) and \(g(x) = 2x^2 - 1\)
a) Find \((f \circ g)(x)\)
\[f(g(x)) = f(2x^2 - 1) = (2x^2 - 1) - 3\]
Answer a)
\[(f \circ g)(x) = 2x^2 - 4\]
b) Find \((g \circ f)(x)\)
\[g(f(x)) = g(x - 3) = 2(x - 3)^2 - 1\]
Expand the binomial
\[= 2(x^2 - 6x + 9) - 1 = 2x^2 - 12x + 18 - 1\]
Answer b)
\[(g \circ f)(x) = 2x^2 - 12x + 17\]
Concept
Domain of a Composite Function
OpenStax
Composite Domain Condition
The domain of the composite function \(f(g(x))\) is the set of all inputs \(x\) such that:
1. \(x\) is in the domain of the inner function \(g\).
2. \(g(x)\) is in the domain of the outer function \(f\).
Step 1 · Inner Restrictions
Identify any values of \(x\) that make the inner function \(g(x)\) undefined. Exclude them.
Step 2 · Outer Restrictions
Look at the final simplified composite expression \(f(g(x))\) and identify any additional values that make it undefined.
Step 3 · Combine
The final domain excludes all values flagged in either Step 1 or Step 2.
Example 5
Find Domain of \(f(g(x))\)
OpenStax
Given
\(f(x) = \dfrac{1}{x+2}\) and \(g(x) = \dfrac{4}{x-1}\)
Step 1 · Find domain of inner function \(g(x)\)
The denominator cannot be zero: \(\;x - 1 \neq 0 \;\Longrightarrow\; x \neq 1\).
Step 2 · Form the composition
\[f(g(x)) = f\left(\dfrac{4}{x-1}\right) = \dfrac{1}{\left(\dfrac{4}{x-1}\right) + 2}\]
Step 3 · Restrict the composite expression
Set denominator \(\neq 0\): \(\;\dfrac{4}{x-1} + 2 \neq 0 \;\Longrightarrow\; \dfrac{4}{x-1} \neq -2\)
Multiply both sides: \(\;4 \neq -2(x - 1) \;\Longrightarrow\; 4 \neq -2x + 2 \;\Longrightarrow\; 2x \neq -2 \;\Longrightarrow\; x \neq -1\).
Step 4 · Combine exclusions
Exclude both \(x = 1\) and \(x = -1\).
Domain: \((-\infty,\, -1) \cup (-1,\, 1) \cup (1,\, \infty)\)

Example 6
Decomposition of Functions
Larson
Strategy
Identify an "inner" function \(g(x)\) (the expression inside parenthesis, roots, exponents, etc.), then write the "outer" function \(f(x)\) that operates on it.
a) Decompose \(h(x) = \sqrt{5x - 4}\)
The inner expression is \(5x - 4\). The outer operation is taking the square root.
Answer a)
\[h(x) = f(g(x)) \quad \text{where} \quad f(x) = \sqrt{x} \quad \text{and} \quad g(x) = 5x - 4\]
b) Decompose \(h(x) = \dfrac{4}{(x+2)^3}\)
The inner expression is \(x + 2\). The outer operation is cubing and dividing \(4\).
Answer b)
\[h(x) = f(g(x)) \quad \text{where} \quad f(x) = \dfrac{4}{x^3} \quad \text{and} \quad g(x) = x + 2\]
Summary
Big Ideas · 1.4
Combined functions
Add, subtract, multiply, or divide outputs. Domain is the overlap of individual domains. Exclude denominator zeroes.
Composition
Nesting functions: \((f \circ g)(x) = f(g(x))\). Work from the inside out. Not commutative!
Composite Domain
Must exclude values that make inner \(g(x)\) undefined AND values that make outer \(f(g(x))\) undefined.
Numerical evaluation
Compute inner output value first, then plug that specific number into the outer function.
Algebraic composition
Replace every variable \(x\) in the outer function \(f(x)\) with the entire expression of \(g(x)\).
Decomposition
Break a complex function into nested pieces. Look for structural groupings to identify \(g(x)\) and \(f(x)\).
